# 依存コマンド追加手順

新しい依存コマンドをAI-DLCに追加する手順。

## 追加手順

1. **env-info.sh**: チェック関数またはcheck_tool呼び出しを追加
2. **setup.md**: 影響説明を追加
3. **各プロンプト**: 利用方法を追加（必要に応じて）

## 汎用ツールの場合

`check_tool` を使用: `echo "newtool:$(check_tool newtool)"`

## 認証が必要なツールの場合

専用のチェック関数を作成（`check_gh` を参考）し、「(ghのみ)」の記述を更新
