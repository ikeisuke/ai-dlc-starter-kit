# Unit: Codex PRレビューIssue Comment承認検出

## 概要
Codex PRレビューの承認判定にIssue Commentパターンを追加し、リアクション判定だけでなくコメントベースの承認も検出可能にする。

## 含まれるユーザーストーリー
- ストーリー 1: Codex PRレビューのIssue Comment承認検出（#408）

## 関連Issue

## 責務
- `rules.md` のc判定ロジックにc-4（Issue Comment判定）を追加
- Codexボットアカウントからの承認コメントの検出パターンを定義
- API失敗時のエラーハンドリング（補助判定として既存判定で続行）

## 境界
- a判定、b判定、c-1/c-2/c-3の既存ロジックは変更しない
- Codex PRレビュー以外のレビューフローは対象外

## 技術的考慮事項
- `rules.md` はプロジェクト固有ルールであり `prompts/package/` 配下ではないため直接編集可能
- Codexボットアカウント定数 `chatgpt-codex-connector[bot]` を再利用
- 承認メッセージの検出: Codexボットが投稿したIssue Commentの本文全体を対象に、既知の承認キーワード（`Didn't find any major issues` 等）を正規表現でマッチ。現時点では単一既知フレーズのみ対象とし、将来的なパターン追加に備えて拡張可能な構造にする
