# Unitブランチ無効化設定

- **発見日**: 2026-01-11
- **発見フェーズ**: Construction
- **発見サイクル**: v1.7.0
- **優先度**: 低

## 概要

Unitブランチ作成を常時無効にする設定を `docs/aidlc.toml` に追加する。

## 詳細

現在、Unitブランチ作成は推奨機能として毎回ユーザーに確認を求めている。
しかし、小規模プロジェクトや個人開発では不要な場合があり、毎回「いいえ」を選択するのは煩雑。

`docs/aidlc.toml` に設定項目を追加し、デフォルトで無効化できるようにする。

## 対応案

```toml
[rules.unit_branch]
enabled = false  # true: 毎回確認, false: 常時無効
```

- `enabled = true`（デフォルト）: 現在の動作（毎回確認）
- `enabled = false`: Unitブランチ作成をスキップ
