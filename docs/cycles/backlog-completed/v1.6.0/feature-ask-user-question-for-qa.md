# 一問一答質問でAskUserQuestionを使用する

- **発見日**: 2025-12-31
- **発見フェーズ**: Construction
- **発見サイクル**: v1.5.3
- **優先度**: 中

## 概要

AI-DLCプロンプトの「予想禁止・一問一答質問ルール」において、Claude Codeを使用している場合は、テキストでの質問ではなくAskUserQuestion機能を使用して質問するように改善する。

## 詳細

現状、AI-DLCプロンプトでは質問時にテキストで質問を表示し、ユーザーの回答を待つ形式を取っている。

Claude CodeにはAskUserQuestion機能があり、選択肢を提示してユーザーに選択させることができる。この機能を活用することで：

- ユーザーの入力負担を軽減
- 選択肢が明確になり、回答のブレが減少
- よりインタラクティブな対話が可能

## 対応案

### 案1: CLAUDE.md に記載（推奨）

プロンプト自体は汎用的に保ち、Claude Code固有の動作はCLAUDE.mdに記載する。

```markdown
# CLAUDE.md

## 質問時のルール

AI-DLCプロンプトで質問する際は、選択肢が明確な場合はAskUserQuestion機能を使用してください。
自由回答が必要な場合はテキストで質問してください。
```

**メリット**:
- プロンプトは汎用的なまま維持
- Claude Code固有の機能はCLAUDE.mdで設定
- 他のAIツールを使う場合も影響を受けない

### 案2: プロンプト内に条件分岐を追加

プロンプト内でClaude Code使用時はAskUserQuestionを使用するよう指示を追加。

**デメリット**: プロンプトがClaude Code固有の記述で複雑になる

## 影響範囲

- `CLAUDE.md`（案1の場合）
- または各プロンプトファイル（案2の場合）
