# 初回セットアップ時のaidlc.toml設定フローを改善

- **発見日**: 2025-12-20
- **発見フェーズ**: サイクル開始時
- **発見サイクル**: v1.5.0
- **優先度**: 中

## 概要

AI-DLC初回セットアップ時のaidlc.toml設定フローを、プロジェクト状況に応じて改善する。

## 詳細

### グリーンフィールド（新規プロジェクト）
README.md などの情報源が少ないため、もっと細かく質問してプロジェクト情報を収集する。

### ブラウンフィールド（既存プロジェクト）
推測できる情報は多いので、可変な設定部分（命名規則、ルール設定など）をどうするかだけ確認する。

## 対応案

- グリーンフィールド: 質問項目を増やしてプロジェクト情報を詳細に収集
- ブラウンフィールド: 可変設定のみ確認するシンプルなフロー
- 両方: デフォルト値やテンプレートを充実させる
