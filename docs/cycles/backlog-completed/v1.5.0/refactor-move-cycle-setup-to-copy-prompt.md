# サイクルセットアップ処理をコピーするプロンプトに移動

- **発見日**: 2025-12-19
- **発見フェーズ**: Inception
- **発見サイクル**: v1.4.1
- **優先度**: 中

## 概要

サイクルセットアップ処理を、コピーするプロンプト（setup-copy.md等）に移動する。

## 詳細

現在、サイクルのセットアップ処理が inception.md 内に含まれているが、これをコピー用のプロンプトに分離することで、責務を明確化する。

## 対応案

- サイクルディレクトリ作成処理を専用プロンプトに移動
- inception.md からはサイクル存在確認のみ行い、存在しない場合は専用プロンプトを案内
