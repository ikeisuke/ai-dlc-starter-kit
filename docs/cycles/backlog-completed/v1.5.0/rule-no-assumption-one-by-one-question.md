# 予想での方針決定禁止・一問一答での質問

- **発見日**: 2025-12-20
- **発見フェーズ**: サイクル開始時
- **発見サイクル**: -
- **優先度**: 高

## 概要

全てのフェーズにおいて、予想で方針を決定しない。不明な点・足りない情報がある場合はユーザーに質問する。質問は事前に全体を示した後、一問一答で行う。

## 詳細

AIが勝手に予想して方針を決めてしまうと、ユーザーの意図と乖離するリスクがある。

質問の進め方:
1. まず質問全体のリストを提示する
2. その後、1問ずつ順番に質問する
3. 各回答を確認してから次の質問に進む

## 対応案

- 各フェーズプロンプトに「予想禁止・質問ルール」を追加
- 質問フローのテンプレートを定義
- 「不明な場合は質問」を明示的にルール化
