# history.mdの複数人開発時コンフリクト対策

- **発見日**: 2025-12-10
- **発見フェーズ**: Operations
- **発見サイクル**: v1.3.0
- **対応サイクル**: v1.4.0
- **完了日**: 2025-12-15
- **優先度**: 低

## 概要

history.mdもheredocで追記する形式のため、複数人が同時に作業するとコンフリクトする

## 対応内容

- history.md をフェーズ/Unit単位のファイル分割方式に変更
- backlog.md を共通バックログに統一、種類別prefix方式に変更
- サイクル固有バックログを廃止
- 完了済みバックログをサイクル別ディレクトリ管理に変更
- 新規テンプレート追加（backlog_item, history_entry）
