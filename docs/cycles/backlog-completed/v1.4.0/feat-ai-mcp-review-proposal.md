# 別のAI MCPが設定されている場合にレビュー活用を提案する

- **発見日**: 2025-12-11
- **発見フェーズ**: セットアップ時
- **発見サイクル**: v1.3.0
- **対応サイクル**: v1.4.0
- **完了日**: 2025-12-14
- **優先度**: 中

## 概要

AI MCPサーバー（例: Codex MCP）が設定されている場合、各フェーズのステップでレビューでの活用を提案する

## 対応内容

- aidlc.toml で MCPレビュー設定を一元管理（mode: "recommend" | "required" | "disabled"）
- 各フェーズプロンプトに MCPレビュー設定を追加
- アップグレード時のマイグレーション機能を追加

## 期待効果

複数AIによるクロスレビューで品質向上、見落とし防止

## 成果物

- Unit 5: AI MCPレビュー推奨機能追加
