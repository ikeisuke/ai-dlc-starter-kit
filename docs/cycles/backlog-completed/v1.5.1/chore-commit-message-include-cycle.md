# コミットメッセージにサイクル名を含める

- **発見日**: 2025-12-20
- **発見フェーズ**: Construction
- **発見サイクル**: v1.5.0
- **優先度**: 低

## 概要

Unit完了時のコミットメッセージにサイクル名を含めるようにする。

## 詳細

現在のコミットメッセージ形式:
```
feat: Unit 001完了 - 予想禁止・一問一答質問ルールを追加
```

希望する形式:
```
feat: [v1.5.0] Unit 001完了 - 予想禁止・一問一答質問ルールを追加
```

## 対応案

- Construction Phase プロンプトの「Unit完了時の必須作業」セクションにあるコミットメッセージ例を更新
- サイクル名 `{{CYCLE}}` をコミットメッセージに含める形式を定義
