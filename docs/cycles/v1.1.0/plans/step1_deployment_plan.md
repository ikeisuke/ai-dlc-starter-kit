## チェックリスト内容（プロンプトファイルプロジェクト向けに簡略化）

### PR作成前チェック
- [ ] すべてのUnit実装が完了している

### PR作成・マージ手順
1. feature/v1.1.0 の最新状態を確認
2. main ブランチとの差分を確認
3. `gh pr create` でPR作成
4. PRレビュー・承認
5. マージ実行

### ロールバック手順
