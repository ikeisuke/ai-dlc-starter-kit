## デプロイ手順

### 1. バージョン確認
```bash
# version.txtの確認
cat version.txt
# 期待値: 1.7.0
```

### 2. ドラフトPRをReady for Reviewに変更
```bash
gh pr ready {PR番号}
gh pr edit {PR番号} --title "v1.7.0"
```

### 3. PRマージ
GitHub上でPRをレビュー・マージ

### 4. 自動タグ作成確認
mainブランチへのマージ後、GitHub Actionsが自動でv1.7.0タグを作成

## ロールバック手順

問題が発生した場合：

```bash
# 前のバージョンに戻す場合
git checkout v1.6.1

# リリースタグを削除（必要な場合）
git tag -d v1.7.0
git push origin --delete v1.7.0
```
