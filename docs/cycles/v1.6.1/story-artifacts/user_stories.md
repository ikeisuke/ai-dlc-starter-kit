## ストーリー1: 共通ルールの分離

**As a** AI-DLC Starter Kit を使用する開発者
**I want** AI-DLC共通ルールが AGENTS.md に、プロジェクト固有ルールが rules.md に分離されている
**So that** 責務が明確になり、メンテナンスが容易になる

### 受け入れ基準
- [ ] AGENTS.md テンプレートに共通ルールセクションが存在する
- [ ] rules_template.md からAI-DLC共通ルールが削除されている
- [ ] AskUserQuestionツールの使用ルール「不明点がなくなるまで繰り返し質問すること」が追加されている
- [ ] フェーズ簡略指示機能が追加されている

### 優先度
高

## ストーリー2: フェーズ簡略指示

**As a** AI-DLCを使用する開発者
**I want** 「コンストラクション進めて」のようなシンプルな指示でフェーズを開始できる
**So that** プロンプトファイルのパスを覚えなくても作業を進められる

### 受け入れ基準
- [ ] 「インセプション進めて」でinception.mdが自動読み込みされる
- [ ] 「コンストラクション進めて」でconstruction.mdが自動読み込みされる
- [ ] ブランチ名からサイクルが自動判定される
- [ ] mainブランチ時はセットアップを促す
- [ ] コンテキストなしで「続けて」の場合はユーザーに確認する

### 優先度
高
