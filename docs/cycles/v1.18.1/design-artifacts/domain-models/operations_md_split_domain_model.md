# ドメインモデル: operations.md分割リファクタリング

## 概要

operations.mdを責務単位で分割する際のファイル境界と参照関係を定義する。

## エンティティ（Entity）

### operations.md（メインプロンプト）

- **属性**:
  - プロジェクト情報セクション（`## プロジェクト情報` 〜 `## あなたの役割` 前）
  - 初期実行ステップセクション（`## 最初に必ず実行すること` 〜 `## フロー` 前）
  - フローセクション: ステップ0〜5（`## フロー` 〜 `### ステップ6` 前）
  - ステップ6リダイレクト（分割後に追加）
  - 実行ルール・完了基準セクション（`## 実行ルール` 〜 `## このフェーズに戻る場合` 末尾）
  - AI-DLCサイクル完了セクション（`## AI-DLCサイクル完了` 〜 ファイル末尾）
- **責務**: Operations Phaseの全体制御とステップ0〜5の詳細手順を提供

### operations-release.md（リリース準備プロンプト）

- **属性**:
  - ファイルヘッダ（見出し + 逆参照 + 実行前提条件）
  - ステップ6の全サブステップ（6.0〜6.7）の詳細手順
- **責務**: リリース準備（ステップ6）の詳細手順を提供
- **実行前提条件**: ステップ0〜5が完了済みであること。共通ルール（`docs/aidlc/prompts/common/rules.md`）および`docs/aidlc.toml`の設定が読み込み済みであること

## 境界

- **分割ポイント**: `### ステップ6: リリース準備` 見出し直前（ステップ5の`- **ステップ完了時**`行の後）
- **分割終了ポイント**: ステップ6セクションの末尾（`---` 区切り）と `## 実行ルール` 見出し直前の間
- **正本のライフサイクル**: operations.md側はナビゲーション用のステップ6概要のみ保持。詳細はoperations-release.mdが正本

## ユビキタス言語

- **正本**: `prompts/package/`配下のファイル（編集対象）
- **コピー**: `docs/aidlc/`配下のファイル（rsyncで同期、直接編集禁止）
- **リダイレクト見出し**: 分割後にoperations.mdに残す、操作の詳細を別ファイルへ誘導する見出し
- **逆参照**: 分割先ファイルから元ファイルへのナビゲーションリンク
