# ドメインモデル: 承認プロンプト頻発の原因調査と対策

## 問題領域

Claude Code の semi_auto モードにおいて、AIが Bash ツールでコマンドを実行する際に承認プロンプトが頻発する。これは開発体験を損ない、semi_auto モードの利便性を低下させる。

## エンティティ

- **承認プロンプト**: Claude Code がユーザーに確認を求めるダイアログ。コマンドの安全性が自動判定できない場合に発生
- **allowedTools設定**: `.claude/settings.json` で許可するコマンドパターンを定義
- **コマンドパターン**: プロンプトファイル内で実行を指示するBashコマンドの形式

## 調査対象

- AI-DLCプロンプトファイル内の全Bashコマンドパターン
- Claude Code の承認判定ロジックとの関係
