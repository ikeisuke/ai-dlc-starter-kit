# Unit: PoC - スキル機能検証

## 概要
スキルからのオンデマンドRead（steps/ファイルのパス解決）とスキル間呼び出し（SkillツールでスキルAからスキルBを呼ぶ）の2つの技術検証を実施し、v2.0.0の実装方針を確定する。

## 含まれるユーザーストーリー
- ストーリー 1: スキル機能の技術検証

## 責務
- テストスキルを作成しプラグインとしてインストール
- オンデマンドRead: SKILL.md内のRead指示でsteps/ファイルを相対パスで読めるか検証
- スキル間呼び出し: テストスキルAからSkillツールでテストスキルBを呼べるか検証
- 検証結果に基づくフォールバック戦略の確定・ドキュメント化

## 境界
- 本番スキルの実装は行わない（テストスキルのみ）
- フェーズプロンプトの分割作業は行わない

## 依存関係

### 依存する Unit
- なし（最初に実施するPoC）

### 外部依存
- Claude Codeのスキル・プラグイン機構

## 非機能要件（NFR）
- **パフォーマンス**: 特になし（検証のみ）
- **セキュリティ**: 特になし
- **スケーラビリティ**: 特になし
- **可用性**: 特になし

## 技術的考慮事項
- パス解決不可の場合は@参照にフォールバック（SKILL.md本文500行制限に注意）
- スキル間呼び出し不可の場合はreviewing-*の内容をsteps/common/review-*.mdにコピー
