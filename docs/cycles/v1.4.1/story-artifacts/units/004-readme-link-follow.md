# Unit: README.mdリンク辿り

## 概要
セットアップ時に README.md のリンク先ドキュメントも読み込むルールを追加する。

## 含まれるユーザーストーリー
- ストーリー3: README.md読み込み時にリンクを辿る

## 責務
- setup-init.md に README.md 読み込み時のリンク辿りルールを追加
- プロジェクト内部のドキュメントリンクを検出・読み込み

## 境界
- 外部リンク（http://、https://）は辿らない
- プロジェクト内部のドキュメントリンクのみ対象

## 依存関係

### 依存する Unit
- なし（他の Unit に依存しない、setup-init.md は独立）

### 外部依存
- なし

## 非機能要件（NFR）
- **パフォーマンス**: リンク辿りによる読み込み増加を考慮
- **セキュリティ**: 該当なし
- **スケーラビリティ**: 該当なし
- **可用性**: 該当なし

## 技術的考慮事項
- 変更対象ファイル: `prompts/setup-init.md`
- コンテキスト溢れ防止のため、辿るリンクの深さや数に制限を設ける
