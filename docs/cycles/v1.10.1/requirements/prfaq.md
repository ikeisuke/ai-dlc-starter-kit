# PRFAQ: AI-DLC Starter Kit v1.10.1

## Press Release（プレスリリース）

**見出し**: AI-DLC Starter Kit v1.10.1 - ドキュメント整備とSkills拡張で開発体験を向上

**副見出し**: README.md並び順修正、PRマージ確認フロー追加、gh/jj Skillsの新規追加により、AI支援開発がさらに使いやすく

**発表日**: 2026-01-29

**本文**:

AI-DLC Starter Kit v1.10.1をリリースしました。本バージョンは、ユーザーからのフィードバックに基づく品質向上と機能拡充を目的としたパッチリリースです。

**解決する課題**:

v1.10.0のリリース後、いくつかの改善点が明らかになりました。README.mdのバージョン一覧の並び順が不統一であること、PRマージ時に意図しないsquashが発生する可能性があること、そしてCodex Skillでの反復レビュー時のresume機能が見落とされがちであることが報告されていました。

**新機能と改善**:

1. **README.mdのバージョン一覧並び順修正** - セマンティックバージョン順（降順）で統一され、最新バージョンが一目でわかるようになりました。

2. **PRマージ確認フロー追加** - Operations Phaseでマージ方法（通常/Squash/Rebase）を明示的に選択できるようになり、意図しないコミット履歴の圧縮を防止できます。

3. **Codex Skill反復レビューresume明確化** - 反復レビュー時のresume使用ガイドが追加され、セッションの継続が効率的に行えるようになりました。

4. **gh (GitHub CLI) Skill新規追加** - Issue操作、PR操作がAIスキルとしてカバーされ、AIとの協調作業でGitHub操作が効率化されます。

5. **jj (Jujutsu) Skill新規追加** - jjの基本操作とgitコマンドとの対照表がスキル化され、モダンなバージョン管理ツールとの連携が強化されます。

## FAQ（よくある質問）

### Q1: v1.10.0からアップグレードする際、何か注意点はありますか？

A: 特別な注意点はありません。通常のアップグレード手順（`prompts/setup-prompt.md`）でアップグレードできます。既存の成果物やプロンプトには影響ありません。

### Q2: gh SkillとjjSkillは必須ですか？

A: いいえ、必須ではありません。これらはオプショナルなSkillsで、gh CLIやjjを使用している場合に活用できます。使用しない場合でも他の機能に影響はありません。

### Q3: PRマージ確認フローは無効化できますか？

A: 現在のバージョンでは無効化オプションはありませんが、常に推奨オプション（通常マージ）を選択することで従来と同じ動作になります。将来のバージョンでデフォルト設定オプションの追加を検討します。

### Q4: Codex Skillのresume機能とは何ですか？

A: Codex CLIのresume機能は、前回のセッション（thread）を継続して使用する機能です。反復レビュー時にresume を使うことで、前回のコンテキストを保持したままレビューを継続でき、より効率的なレビューが可能になります。
