# Unit 002 計画: PRマージ確認フロー追加

## 概要

Operations Phaseのステップ6.5にPRマージ方法の選択フローを追加する。

## 変更対象ファイル

- `prompts/package/prompts/operations.md`

## 実装計画

### 変更内容

ステップ6.5「ドラフトPR Ready化」の「Ready化成功時」メッセージの後に、マージ方法選択フローを追加する。

**挿入位置**: 584-585行目（「レビューを依頼し、マージしてください。」）を、マージ方法選択フローに置き換える。

**追加するフロー**:

1. マージ方法の確認を提示
   - 通常マージ（デフォルト）
   - Squashマージ / Rebaseマージ（明示的に指示があった場合のみ）

2. デフォルト動作の説明
   - 特別な指示がない限り、通常マージで実行する
   - ユーザーが「squash」「rebase」と明示した場合のみ該当方法を使用

3. 通常マージの`gh pr merge`コマンドを実行

### 設計省略の理由

このUnitはプロンプトファイル（ドキュメント）の修正のみであり、アプリケーションコードの実装を伴わないため、ドメインモデル設計・論理設計は省略する。
