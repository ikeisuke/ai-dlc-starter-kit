# Unit 003 計画: Codex Skill反復レビューresume明確化

## 概要

Codex Skillの説明に反復レビュー時のresume使用ガイドを追加し、効率的なレビュー継続を支援する。

## 変更対象ファイル

| ファイル | 変更内容 |
|----------|----------|
| `prompts/package/skills/codex/SKILL.md` | 「反復レビュー時のルール」セクション追加 |

## 実装計画

### Phase 1: 設計（省略）

本Unitはドキュメント追加のみのため、ドメインモデル・論理設計を省略する。

### Phase 2: 実装

1. **セクション追加位置の決定**
   - 既存の「セッション継続（resume）」セクションの後に配置
   - または「使用例」セクションの前に配置

2. **「反復レビュー時のルール」セクションの作成**
   - session id の確認・記録方法の明記
   - resumeを使うべき場面の強調
   - 反復レビューの流れの例示
   - review-flow.md との整合性確認

3. **内容の検証**
   - Codex CLIの実際の動作と整合性があることを確認
