# Unit 005 計画: jjサポート強化

## 概要

`jj-support.md` を強化し、よくあるミスと対処法を追加する。

## 変更対象ファイル

- `prompts/package/guides/jj-support.md`（編集 ※`docs/aidlc/guides/` はrsyncコピーのため直接編集禁止）

## 実装計画

### Phase 1: 設計

1. **ドメインモデル設計**: 追加する「よくあるミスと対処法」の構造を定義
2. **論理設計**: 既存のドキュメント構成との整合性を確認し、配置位置を決定
3. **設計レビュー**: ユーザー承認

### Phase 2: 実装

1. **「よくあるミスと対処法」セクションの追加**
   - `jj new` 忘れの対処法
   - bookmarkが進まない問題の詳細な対処法
   - 不要ブランチ（bookmark）の削除方法
   - 作業開始前に未コミットファイルがある場合の対処法
   - その他の実践的なケース

2. **コマンド対照表の補完**（必要に応じて）
   - 既存の対照表を確認し、不足コマンドがあれば追加

3. **統合とレビュー**
   - 全体の整合性確認
   - ドキュメントの動作確認
