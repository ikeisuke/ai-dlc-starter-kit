# Unit: セットアップスクリプト修正

## 概要

セットアップスクリプトを修正し、スキルファイルのコピーとシンボリックリンク作成を自動化する。

## 含まれるユーザーストーリー

- ストーリー 1-1: プロジェクトローカルスキルの自動利用（後半）

## 責務

- rsync対象に `skills/` ディレクトリを追加
- `.claude/skills/` から `docs/aidlc/skills/` へのシンボリックリンク作成処理を追加
- 既存のシンボリックリンクがある場合の処理（上書き or スキップ）

## 境界

- スキルファイルの内容変更は含まない（Unit 001で対応済み）
- KiroCLI対応は含まない（Unit 003で対応）

## 依存関係

### 依存する Unit

- Unit 001: スキルファイル配置（依存理由: コピー元のスキルファイルが必要）

### 外部依存

- rsync コマンド
- ln コマンド（シンボリックリンク作成）

## 非機能要件（NFR）

- **パフォーマンス**: 特になし
- **セキュリティ**: 特になし
- **スケーラビリティ**: 新しいスキル追加時に自動で対応（ディレクトリごとリンク）
- **可用性**: 特になし

## 技術的考慮事項

- シンボリックリンクは相対パスを使用（環境依存を排除）
- Gitで追跡可能なシンボリックリンク形式
- 冪等性を確保（再実行しても問題ない）

High
