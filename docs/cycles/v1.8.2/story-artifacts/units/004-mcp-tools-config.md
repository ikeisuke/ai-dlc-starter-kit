# Unit: AIレビュー設定強化

## 概要

`aidlc.toml` の `[rules.mcp_review]` セクションに `mcp_tools` 設定を追加し、MCPツールの優先順位を設定可能にする。

## 含まれるユーザーストーリー

- ストーリー 2-1: MCPツールの設定指定
- ストーリー 2-2: フォールバック動作

## 責務

- `aidlc.toml` に `mcp_tools` 設定を追加
- 各フェーズのプロンプトを修正（設定を読み取り、優先順位に従ってツールを選択）
- フォールバック動作の実装（すべて利用不可 → Skills）

## 境界

- 新しいMCPツールの追加は含まない
- Skills機能自体の変更は含まない

## 依存関係

### 依存する Unit

- なし（独立して実装可能）

### 外部依存

- MCPツール（`mcp__codex__codex` 等）
- Skill `codex`

## 非機能要件（NFR）

- **パフォーマンス**: ツール利用可否チェックは高速に行う
- **セキュリティ**: 設定ファイルの読み取りのみ
- **スケーラビリティ**: 新しいMCPツール追加時にリスト追加で対応可能
- **可用性**: フォールバックにより常にレビューが実行可能

## 技術的考慮事項

- 設定形式: `mcp_tools = ["mcp__codex__codex", "mcp__other__tool"]`
- 空の場合のデフォルト動作: 現状維持（Skills優先 → MCP `mcp__codex__codex`）
- 既存の `mode` 設定との互換性を維持
