# Unit: その他のAIツール対応ドキュメント

## 概要

Claude Code以外のAIツール（Codex、Gemini、KiroCLI等）でAIスキルを利用する方法をドキュメント化する。

## 含まれるユーザーストーリー

- ストーリー 1-2: KiroCLI対応ドキュメント
- （拡張）各AIツールでのスキル参照方法

## 責務

- 各AIツールでのスキル参照方法を文書化
  - Codex CLI: プロジェクト内ファイルの直接参照
  - Gemini: MCP経由でのファイル読み取り
  - KiroCLI: `resources` フィールドへのスキル追加例
- AGENTS.md の各ツール対応セクションを更新
- 必要に応じてスキルファイル内に各ツール向けの説明を追加

## 境界

- 各AIツール自体の機能開発は含まない
- Claude Code固有の設定は含まない（Unit 002で対応済み）

## 依存関係

### 依存する Unit

- Unit 001: スキルファイル配置（依存理由: 参照するスキルファイルのパスが確定している必要がある）
- Unit 002: セットアップスクリプト修正（依存理由: skills/がdocs/aidlc/に同期されている必要がある）

### 外部依存

- KiroCLIのドキュメント（参照用）
- Codex CLIのドキュメント（参照用）

## 非機能要件（NFR）

- **パフォーマンス**: 特になし
- **セキュリティ**: 特になし
- **スケーラビリティ**: 新しいAIツール追加時に拡張可能な構成
- **可用性**: 特になし

## 技術的考慮事項

- KiroCLIは `@ファイルパス` 記法が機能しないことに注意
- 各ツールの制約に応じた参照方法を明確に説明
- ツールごとの設定例を具体的に記載
