# 実装記録: 環境情報一覧スクリプト（Unit 001）

## 概要

依存ツール（gh, dasel, jj, git）の状態を一覧で出力するスクリプトを実装した。

## 実装内容

### 作成ファイル

| ファイル | 説明 |
|----------|------|
| `prompts/package/bin/env-info.sh` | 環境情報一覧スクリプト |

### 機能

- 4つの依存ツール（gh, dasel, jj, git）の状態を確認
- 統一フォーマット（`ツール名:状態`）で出力
- gh固有の認証状態確認
- ヘルプオプション（`-h`, `--help`）のサポート
- 不正オプション時のエラーハンドリング

### 出力例

```text
gh:available
dasel:available
jj:available
git:available
```
