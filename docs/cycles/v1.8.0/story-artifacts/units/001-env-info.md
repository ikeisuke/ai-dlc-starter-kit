# Unit: 環境情報一覧スクリプト

## 概要
gh, dasel, jj等の依存ツールの状態を一覧で出力するスクリプトを作成する。

## 含まれるユーザーストーリー
- ストーリー 1-1: 環境情報一覧の出力

## 関連Issue

## 責務
- 依存ツール（gh, dasel, jj, git）の存在確認
- 各ツールの状態を統一フォーマットで出力
- プロンプト内の環境確認をスクリプト呼び出しに置換

## 境界
- ツールのインストールは行わない（確認のみ）
