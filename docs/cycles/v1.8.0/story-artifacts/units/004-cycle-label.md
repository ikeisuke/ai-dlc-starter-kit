# Unit: サイクルラベル操作スクリプト

## 概要
サイクルラベル（cycle:vX.X.X）の確認と作成を一括で行うスクリプトを作成する。

## 含まれるユーザーストーリー
- ストーリー 1-4: サイクルラベル操作

## 関連Issue
- #34

## 責務
- サイクルバージョンを引数で受け取る
- ラベル存在確認
- 存在しなければ作成
- プロンプト内のラベル確認+作成をスクリプト呼び出しに置換

## 境界
- 共通ラベル（backlog, type:*等）は別Unitで対応

## 依存関係

### 依存する Unit
- Unit 001（環境情報でgh確認）

### 外部依存
- gh（GitHub CLI）

- **セキュリティ**: N/A
- **スケーラビリティ**: N/A
