# Unit: バックログ対応済みチェック

## 概要
Inception Phaseでバックログを確認する際に、過去に対応済みの項目かどうかを自動チェックする機能を追加する。

## 含まれるユーザーストーリー
- ストーリー1: バックログ対応済みチェック

## 責務
- バックログ確認時にbacklog-completed.mdの内容と照合する
- 対応済みの項目があればユーザーに通知する

## 境界
- バックログ項目の自動移動は行わない（通知のみ）
- 過去サイクルのhistory.md検索は行わない（backlog-completed.mdのみ参照）
