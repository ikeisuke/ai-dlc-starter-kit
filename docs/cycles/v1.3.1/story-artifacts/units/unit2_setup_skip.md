# Unit: セットアップスキップ

## 概要
アップグレード不要時にセットアップをスキップしてInception Phaseを直接開始できるようにする。サイクルディレクトリの自動作成と最新バージョン通知を含む。

## 含まれるユーザーストーリー
- ストーリー2: セットアップスキップ（サイクル自動作成）
- ストーリー3: 最新バージョン通知

## 責務
- サイクルディレクトリが存在しない場合、自動作成を提案する
- ユーザー承認後、サイクルディレクトリを作成する
- 最新バージョンが利用可能な場合、通知する

## 境界
- アップグレードの強制は行わない（通知のみ）
- setup-cycle.mdの既存フローは変更しない
