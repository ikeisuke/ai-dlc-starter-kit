# Unit 002: jjサポート - 設定とドキュメント ドメインモデル

## 概要

このUnitはドキュメント修正が主体のため、ドメインモデルは簡略化して記載する。

## ドメイン概念

### 1. jj設定 (JjConfiguration)

**責務**: jjサポートの有効/無効を制御する設定

**属性**:
- enabled: boolean - jjサポートの有効化フラグ（デフォルト: false）

**配置場所**: `docs/aidlc.toml` の `[rules.jj]` セクション

### 2. jjガイド (JjGuide)

**責務**: jjの使い方とGitとの違いを説明するドキュメント

**構成要素**:
- 概要セクション
- 前提条件セクション
- **gitとjjの考え方の違いセクション** ← 今回追加
- jjの特徴と利点セクション
- Git/jjコマンド対照表セクション
- AI-DLCワークフローでの使用方法セクション
- 注意事項と制限セクション
- 参考リンクセクション
