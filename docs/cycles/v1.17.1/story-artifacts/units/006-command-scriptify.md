# Unit: $()コマンドスクリプトファイル化

## 概要
prompts/package/配下のプロンプトで使用されている`$(ghq root)`等の$()コマンドをスクリプトファイル化し、Claude Codeでの実行時の許可確認を不要にする。

## 含まれるユーザーストーリー
- ストーリー 6: $()コマンドのスクリプトファイル化 (#243)

## 関連Issue
- #243

## 責務
- setup-prompt.md内で使用されている`$(ghq root)`等の$()コマンドのスクリプトファイル化
- スクリプトの`prompts/package/bin/`への配置
- スクリプトの出力と終了コードがスクリプト化前のコマンドと同等であることの確認
- setup-prompt.md内のコマンド例のスクリプト呼び出しへの更新

## 境界
- setup-prompt.md以外のプロンプトの$()コマンドは含まない
- 新規機能の追加は含まない（既存コマンドの置き換えのみ）
