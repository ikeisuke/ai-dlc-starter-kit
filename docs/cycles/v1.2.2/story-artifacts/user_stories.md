# ユーザーストーリー（簡易版）

## US-1: 気づき記録フロー
- **As a** AI-DLC利用者
- **I want to** Unit作業中に別Unitに関する気づきがあった場合の対応方法を知りたい
- **So that** 作業の集中を維持しつつ、気づきを失わない

## US-2: ホームディレクトリ共通設定
- **As a** 複数プロジェクトでAI-DLCを使う開発者
- **I want to** ホームディレクトリに共通設定を配置したい
- **So that** プロジェクト横断で一貫した設定を使える

## US-3: ファイルコピー判定改善
- **As a** AI-DLCセットアップを実行する開発者
- **I want to** ファイルコピー時に変更有無をハッシュ値で判定してほしい
- **So that** 不要なコピーや意図しない上書きを防げる

## US-4: Lite版の案内追加
- **As a** 新規サイクルを開始する開発者
- **I want to** セットアップ完了時にLite版の選択肢も見たい
- **So that** 軽量な修正にはLite版を選択できる

## US-5: サイクル固有バックログ確認
- **As a** Inception Phaseを実行するAI
- **I want to** サイクル固有バックログも確認したい
- **So that** そのサイクルで発見された気づきも漏れなく対応できる
