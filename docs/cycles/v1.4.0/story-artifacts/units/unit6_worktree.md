# Unit 6: git worktree提案

## 概要
セットアップ時にgit worktreeの使用を提案し、複数サイクルの並行作業を支援する

## 含まれるユーザーストーリー

## 責務
- セットアップ時にworktreeの使用オプションを提案する
- worktreeのディレクトリ構成を案内する
- 通常のブランチ切り替えも選択可能にする

## 境界

- worktreeの自動作成は扱わない（提案のみ）
