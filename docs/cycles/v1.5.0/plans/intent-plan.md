# Intent作成計画

**作成日**: 2024-12-20
**ステップ**: 1. Intent明確化

## 作成予定のIntent

### プロジェクト名
AI-DLC スターターキット v1.5.0

### 開発の目的
AI-DLCプロンプトの品質向上とルール明確化を行う。具体的には:
- AIの予想による方針決定を禁止し、一問一答での質問ルールを導入
- Construction Phase以外でのコード記述を制限
- 外部入力（AI MCP・ユーザー）の検証ルールを追加
- サイクルセットアップ処理を専用プロンプトに分離
- Operations Phaseのセルフアップデート処理を廃止
- グリーンフィールドセットアップ時の質問フローを改善

### ターゲットユーザー
AI-DLCスターターキットを使用する開発者

### ビジネス価値
- AIの自律的判断によるリスクを低減
- フェーズごとの責務を明確化
- セットアップ体験の向上

### 成功基準
1. 6件のバックログ項目がすべてプロンプトに反映されている
2. 新ルール（予想禁止・コード記述制限・外部入力検証）が各フェーズプロンプトに追加されている
3. サイクルセットアップが専用プロンプトに分離されている
4. Operations Phase からセルフアップデート処理が削除されている

### 期限とマイルストーン
特になし（サイクル完了時点で達成）

### 制約事項
- **メタ開発ルール厳守**: `docs/aidlc/` は直接編集禁止、必ず `prompts/package/` を編集する
- プロンプト変更は次回セットアップ時に反映される
