# Unit: Issueテンプレート差分確認スクリプト

## 概要
ローカルとリモートのGitHub Issueテンプレートの差分を自動検出するスクリプトを作成する。

## 含まれるユーザーストーリー
- ストーリー 3: Issueテンプレート差分確認のスクリプト化 (#205)

## 責務
- `check-issue-templates.sh` の新規作成
- `.github/ISSUE_TEMPLATE/` 配下のローカルとリモート（mainブランチ）の差分検出
- gh未インストール・未認証・ネットワークエラー時のエラーハンドリング

## 境界
- テンプレートの自動修正・同期は行わない（差分検出のみ）
- GitHub Actions等のCI連携は行わない

## 依存関係

### 依存する Unit
- なし

### 外部依存
- GitHub CLI（gh）
- GitHub API

## 非機能要件（NFR）
- **パフォーマンス**: 通常ネットワーク条件下で目安10秒以内（GitHub APIレスポンスに依存）
- **セキュリティ**: 該当なし
- **スケーラビリティ**: 該当なし
- **可用性**: gh未インストール時に明確なエラーメッセージを出力

## 技術的考慮事項
- `gh api` でリモートファイルを取得
- `diff` コマンドで差分検出
- 既存スクリプト（check-gh-status.sh）のパターンに準拠
