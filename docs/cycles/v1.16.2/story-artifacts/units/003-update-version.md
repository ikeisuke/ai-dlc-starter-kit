# Unit: バージョンファイル更新スクリプト

## 概要
version.txtとaidlc.tomlのバージョン番号を1コマンドで一括更新するスクリプトを作成する。

## 含まれるユーザーストーリー
- ストーリー 4: バージョンファイル更新のスクリプト化 (#204)

## 責務
- `update-version.sh` の新規作成
- サイクル番号からvプレフィックスを除去してversion.txtを更新
- docs/aidlc.tomlのstarter_kit_versionを同じ値に更新
- 引数バリデーション（未指定、不正フォーマット）
- --dry-runオプション対応

## 境界
- version.txt / aidlc.toml 以外のファイルのバージョン更新は行わない
- iOSプロジェクト固有のバージョン更新は行わない

## 依存関係

### 依存する Unit
- なし

### 外部依存

- **セキュリティ**: 該当なし
- **スケーラビリティ**: 該当なし
- **可用性**: 該当なし

## 技術的考慮事項
- macOS互換の `sed -i ''` 形式を使用
- `rules.md` の手動手順（「バージョンファイル更新【重要】」セクション）をスクリプト呼び出しに置き換え
- 既存スクリプトの出力形式（`key:value`）に準拠
