# Unit: rsync同期スクリプト

## 概要
prompts/package/からdocs/aidlc/へのrsync同期を1コマンドで実行するスクリプトを作成する。

## 含まれるユーザーストーリー
- ストーリー 5: rsync同期処理のスクリプト化 (#203)

## 責務
- `sync-package.sh` の新規作成
- prompts/package/ → docs/aidlc/ のrsync同期実行
- 除外パターン設定（.git, .DS_Store）
- 同期結果（追加・更新・削除ファイル一覧）の出力
- --dry-runオプション対応
- ソース/宛先ディレクトリ不存在時のエラーハンドリング

## 境界
- rsync以外の同期方式は検討しない
- 双方向同期は行わない（prompts/package/ → docs/aidlc/ の一方向のみ）

## 依存関係

### 依存する Unit
- なし

### 外部依存
- rsync（macOS/Linux標準）

## 非機能要件（NFR）
- **パフォーマンス**: 数秒以内に同期完了（ファイル数に依存）
- **セキュリティ**: 該当なし
- **スケーラビリティ**: 該当なし
- **可用性**: 該当なし
