## 改善内容（Issue #49より）

### 短期対応（本Unitで実施）
1. **ドキュメント改善**
   - `docs/aidlc/guides/jj-support.md` に「作業開始時」「作業終了時」セクションを追加
   - 各セクションにチェックリストとコマンド例を記載
   - AI-DLCプロンプト内でjj操作を明示的にガイド

2. **jj設定の推奨**
   - `docs/aidlc/guides/jj-support.md` に推奨設定を追記
   ```toml
   # .jj/config.toml
   [git]
   auto-local-bookmark = true
   ```
