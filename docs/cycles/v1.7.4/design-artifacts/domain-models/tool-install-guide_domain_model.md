# ドメインモデル: ツールインストール案内

## 概要

このUnitはドキュメント追加のみのため、ドメインモデルは最小限の記述とする。

## ドメイン概念

### ツール（Tool）

AI-DLCの実行に必要なコマンドラインツール。

| 属性 | 説明 |
|------|------|
| name | ツール名（gh, dasel, jq, curl） |
| required | 必須かオプションか |
| purpose | 使用目的 |
| installCommand | インストールコマンド（パッケージマネージャー別） |
| alternative | オプションツールの場合、代替手段 |

### パッケージマネージャー（PackageManager）

ツールのインストールに使用するパッケージマネージャー。

| 対象 | 名前 |
|------|------|
| macOS | Homebrew (brew) |
| Ubuntu/Debian | APT (apt) |

## ツール一覧

| ツール | 必須/オプション | 用途 | 代替手段 |
|--------|----------------|------|----------|
| gh | 必須 | GitHub Issue/PR操作、バックログ管理 | なし |
| dasel | オプション | TOML設定ファイルの解析 | AIが直接ファイルを読み取り |
| jq | オプション | JSON解析 | AIが直接解析 |
| curl | オプション | Webリソース取得 | AIのWebFetch機能 |
