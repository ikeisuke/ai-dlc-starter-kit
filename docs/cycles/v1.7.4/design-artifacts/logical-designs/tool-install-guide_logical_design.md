`prompts/setup-prompt.md` に「必要ツールの準備」セクションを追加する。

## 変更対象

- **ファイル**: `prompts/setup-prompt.md`
- **変更種類**: セクション追加

## セクション配置

```text
# AI-DLC セットアップ

（既存の導入文）

---

（ツールインストール案内）

---

## 1. 実行環境の確認   ← 既存（番号変更なし）
```

## セクション構成

### 0. 必要ツールの準備

1. **導入文**: AI-DLCの実行に必要なツールの説明
2. **必須ツール**: gh（GitHub CLI）のインストール案内
3. **オプションツール**: dasel, jq, curl の説明（AIが代替可能な旨を明記）
4. **インストールコマンド**: brew / apt のコマンド例
