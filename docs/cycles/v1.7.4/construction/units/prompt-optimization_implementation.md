## 変更詳細

### 1. AGENTS.md

「AI-DLC共通ルール」セクションに「質問と深掘り」を追加:

- 質問の目的（曖昧な要件の明確化、前提条件確認、意図の特定）
- 深掘りテクニック（「具体的には？」「例えば？」「なぜ？」）
- 注意事項（ハイブリッド方式に従い、概要提示後は一問一答）

### 2. inception.md

ステップ3「ユーザーストーリー作成」に受け入れ基準ガイダンスを追加:

- 良い例（具体的で検証可能な基準）
- 悪い例（曖昧で検証困難な基準）
- 記述のポイント（主語・動詞・結果の明確化、テスト可能な形式）

また、ステップ1の一問一答形式の記述を「質問の概要を先に提示した後は、1つの質問をして回答を待つ（ハイブリッド方式に従う）」に修正。
