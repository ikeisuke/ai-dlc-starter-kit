# Unit 007 計画: CI/CD構築

**作成日**: 2025-12-31
**対象Unit**: 007-cicd-setup

## 目的

GitHub ActionsでCI/CDパイプラインを構築し、PRに対してMarkdownリンターを自動実行する。

## 実装内容

### 1. GitHub Actionsワークフロー

**ファイル**: `.github/workflows/pr-check.yml`

- PRが作成・更新されたときに自動実行
- mainブランチへのPRを対象
- Markdownリンターを実行

### 2. Markdownリンター設定

**ファイル**: `.markdownlint.json`

- プロジェクト固有のルールを設定
- AI-DLCのドキュメント構造に合わせた設定

### 3. 対象ディレクトリ

リンター対象:
- `docs/` - ドキュメント全般
- `prompts/` - プロンプトファイル

## 実行計画

### Phase 1: 設計

1. **ドメインモデル設計**: CI/CDパイプラインの概念モデル
2. **論理設計**: ワークフロー構成とリンター設定の詳細
3. **設計レビュー**: ユーザー承認

### Phase 2: 実装

1. **コード生成**: ワークフローファイルとリンター設定ファイルを作成
2. **検証**: ローカルでリンターを実行して動作確認
