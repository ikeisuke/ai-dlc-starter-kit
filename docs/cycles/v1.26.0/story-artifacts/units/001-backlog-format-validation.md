# Unit: バックログモード別フォーマットバリデーション

## 概要
バックログモード（git, issue, git-only, issue-only）に応じた成果物形式の排他的バリデーションルールを `rules.md` に追加し、AIエージェントの誤動作を防止する。

## 含まれるユーザーストーリー
- ストーリー 1: バックログモード別の成果物フォーマットバリデーション

## 関連Issue
- #315

## 責務
- 各バックログモードで許可・禁止される成果物形式をルールとして明文化
- 禁止形式での保存時の期待動作（警告出力・正しい形式での保存し直し）を定義

## 境界
- スクリプトの変更は含まない（プロンプトテキストのみ）
- 新規バックログモードの追加は含まない
- バックログ管理ガイド（`docs/aidlc/guides/backlog-management.md`）の大幅改修は含まない
