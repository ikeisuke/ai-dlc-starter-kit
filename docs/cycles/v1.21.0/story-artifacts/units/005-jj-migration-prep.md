# Unit: jjスキル移行準備

## 概要
jjスキルの移行手順ドキュメントを作成し、skillsリポジトリへの追加Issueを作成する。ファイル退避はUnit 004で実施済みのため、本Unitはドキュメント作成とIssue作成に特化する。

## 含まれるユーザーストーリー
- ストーリー 6: jjスキルのskillsリポジトリへの移動

## 責務
- 移行手順ドキュメント（`docs/aidlc/guides/jj-migration.md`）の作成
  - skillsリポジトリへのインストール手順（マーケットプレイス方式）
  - 手動インストール手順（マーケットプレイス未使用の場合）
- Unit 004で追加した `aidlc-setup.sh` の暫定移行案内文言を、`jj-migration.md` への参照に更新
- skillsリポジトリへの追加 Issue の作成

## 境界
- ファイル退避はUnit 004で実施済み（本Unitでは行わない）
- skillsリポジトリへのPR作成・マージは行わない（別作業）
- jjスキルの機能拡張は行わない

## 依存関係

### 依存する Unit
- 004-remove-jj-code（依存理由: 退避ファイルがUnit 004で作成されるため）
- 002-marketplace-support（依存理由: マーケットプレイス方式の移行手順を書くため、実装実態が確定している必要がある）

### 外部依存
- skillsリポジトリ（Issue作成先）

## 非機能要件（NFR）
- **パフォーマンス**: 該当なし
- **セキュリティ**: 該当なし
- **スケーラビリティ**: 該当なし
- **可用性**: 該当なし

## 技術的考慮事項
- 本サイクルのスコープは本リポジトリ側の作業に限定
- 移行手順ドキュメントの編集元パス: `prompts/package/guides/jj-migration.md`、同期後パス: `docs/aidlc/guides/jj-migration.md`
- Unit 004で追加した `aidlc-setup.sh` の暫定移行案内文言を、本Unitで `jj-migration.md` への参照に更新する
