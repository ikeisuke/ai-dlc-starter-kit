# Unit: マルチプラットフォーム対応調査

## 概要
AIエージェント別の対応状況を体系的に調査し、対応状況マトリクス・ギャップ分析・優先対応提案を含む調査ドキュメントを作成する。

## 含まれるユーザーストーリー
- ストーリー 7: マルチプラットフォーム対応状況の調査・文書化

## 責務
- 対応状況マトリクス（AIエージェント × 機能）の作成
  - 対象エージェント: Claude Code, KiroCLI, Codex CLI, Gemini CLI, Cursor, Cline, Windsurf
  - 対象機能: 設定ファイル, スキル連携, コミット属性, レビュー, サブエージェント, Plan Mode
- ギャップ分析（未対応・部分対応項目の不足内容）
- Claude Code固有表現の使用箇所一覧
- 次期サイクルへの優先対応提案（優先度と理由付き）

## 境界
- 調査・文書化のみ。マルチプラットフォーム対応の実装は行わない
- 新規設定ファイル（.cursorrules等）の作成は行わない
