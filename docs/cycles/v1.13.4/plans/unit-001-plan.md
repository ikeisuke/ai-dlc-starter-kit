# Unit 001 計画: Codex skillsの設定ドキュメント作成

## 概要

Codex CLIで AI-DLCスキルを利用するための設定方法を `skill-usage-guide.md` に追記する。
Codex CLIは `~/.codex/skills` からスキルファイルを読み込む仕様のため、手動でシンボリックリンクを作成する手順を文書化する。

## 変更対象ファイル

| ファイル | 変更内容 |
|---------|----------|
| `prompts/package/guides/skill-usage-guide.md` | Codex CLI用スキル設定セクションを追加 |

## スコープ外

- セットアップスクリプトの自動化は行わない
- Codex CLI自体のインストール手順は含まない

## 実装計画

### Phase 1: 設計

1. **論理設計**: 追記するセクションの構成・配置を定義（ドキュメント変更のみのためドメインモデルは簡略化）
2. **設計レビュー**: AIレビュー + ユーザー承認

### Phase 2: 実装

3. **コード生成**: `prompts/package/guides/skill-usage-guide.md` に Codex CLI スキル設定セクションを追記
   - `~/.codex/skills` へのシンボリックリンク作成手順
   - 設定後の確認方法（`ls ~/.codex/skills/` でシンボリックリンク存在確認）
   - 既存セットアップフロー（`prompts/setup-prompt.md`）との関係説明
4. **テスト生成**: ドキュメント変更のためコードテストは不要。Markdownlintによる形式チェックのみ
5. **統合とレビュー**: AIレビュー + ユーザー承認
