# Unit: Operations Phaseスキップ判定改善

## 概要

Operations Phaseのステップ0「変更確認」のスキップ判定を改善する。バージョン確認をステップ1からステップ6に移動し、ステップ5（バックログ整理）を常に実行するようにする。

- バージョン確認ロジック（iOS判定、ビルド番号確認、通常バージョン確認）をステップ1からステップ6の冒頭に移動

## 境界
