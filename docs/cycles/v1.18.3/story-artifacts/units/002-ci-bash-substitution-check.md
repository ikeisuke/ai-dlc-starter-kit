# Unit: CI Bashコードブロック コマンド置換検出

## 概要
プロンプト内のBashコードブロックで `$()` およびバッククォート（`` ` ``）によるコマンド置換が使用されていないことをCIで自動検出する仕組みを導入する。

## 境界
- 説明文やインラインコード内の `$()` やバッククォートは対象外
- `.sh` スクリプトファイルは対象外
- 検出対象は `prompts/package/prompts/**/*.md` のみ
