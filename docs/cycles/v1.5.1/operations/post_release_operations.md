# リリース後の運用計画

## リリース情報
- **バージョン**: v1.5.1
- **リリース予定日**: 2025-12-22
- **リリース内容**: セットアップ体験とプロンプト構成の改善

## 今回のサイクルの主要変更

1. **プロジェクトタイプ設定機能** - 初回セットアップ時と設定未設定時にプロジェクトタイプを確認
2. **履歴記録設定機能** - 履歴記録のレベルを設定可能に（detailed/standard/minimal）
3. **コミットメッセージ改善** - サイクル名を含める形式に変更
4. **セットアップエントリーポイント変更** - 通常サイクルは `docs/aidlc/prompts/setup.md` を使用
5. **セットアッププロンプト統合** - 責務分離と導線の明確化
