# デプロイチェックリスト

- **デプロイ方式**: GitHubリポジトリ公開（mainブランチへのマージ + 自動タグ作成）

## デプロイ前チェックリスト

### バージョン管理
- [x] `version.txt` がサイクルバージョンと一致している（1.5.1）
- [ ] README.md が更新されている

- [ ] すべての変更がコミットされている

### ドキュメント
- [ ] 今回のサイクルの変更内容が記載されている

## デプロイ手順

### 1. 事前確認
```bash
# ブランチ確認
git branch

# 未コミットの変更確認
git status

# version.txt確認
cat version.txt
```

### 2. Operations Phase完了コミット
```bash
git add .
git commit -m "chore: [v1.5.1] Operations Phase完了"
```

### 3. PR作成
```bash
gh pr create --base main --title "v1.5.1" --body "..."
```

### 4. PRマージ後
- GitHub Actionsが自動で `v1.5.1` タグを作成

## ロールバック手順

問題が発生した場合：
```bash
# 前のバージョンに戻す場合
git checkout v1.5.0
```
