# Unit: プロジェクトタイプ設定機能の追加

## 概要
初回セットアップ時にプロジェクトタイプを選択・保存し、Operations Phaseで参照できるようにする

## 含まれるユーザーストーリー
- ストーリー1: プロジェクトタイプの設定

## 責務
- setup-init.md にプロジェクトタイプ選択ステップを追加
- aidlc.toml テンプレートに project.type フィールドを追加
- operations.md で project.type を参照し、配布ステップの要否を判断

## 境界
- プロジェクトタイプに応じた具体的な配布処理は対象外（既存のまま）
