# Unit 004 計画: サンドボックス環境ガイド

## 概要

AIエージェントをサンドボックス環境で安全に実行する方法を説明するガイドを作成する。

## 変更対象ファイル

| ファイル | 操作 | 説明 |
|---------|------|------|
| `prompts/package/guides/sandbox-environment.md` | 新規作成 | サンドボックス環境ガイド |

## 実装計画

### Phase 1: 設計

1. **ドメインモデル設計**: ガイドの構成と各セクションの責務を定義
2. **論理設計**: 各AIエージェントのサンドボックス設定の調査・整理
3. **設計レビュー**: ユーザー承認

### Phase 2: 実装

4. **コード生成**: `sandbox-environment.md` ファイルを作成
   - 概要セクション: サンドボックスの目的と利点
   - Claude Code: Docker環境での実行方法、`--dangerously-skip-permissions` の説明
   - Codex CLI: `sandbox` 設定オプションの説明
   - KiroCLI: サンドボックス設定の説明（未対応の場合は明記）
   - Docker/コンテナ: 汎用的なコンテナ環境での実行例
   - セキュリティ注意事項: リスクと推奨設定
5. **テスト生成**: N/A（ドキュメントのみ）
6. **統合とレビュー**: AIレビュー実施
