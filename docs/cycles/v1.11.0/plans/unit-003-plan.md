# Unit 003: AIレビュー完了条件明示化 - 実装計画

## 概要

AIレビュー完了時に明示的な完了メッセージを出力し、再レビュー漏れを防止する。
指摘0件の場合のみ完了メッセージを出力し、指摘がある場合は修正→再レビューのフローを維持する。

## 変更対象ファイル

| ファイル | 変更内容 |
|----------|----------|
| `prompts/package/prompts/common/review-flow.md` | AIレビュー完了メッセージ出力指示を追加 |

## 実装計画

### Phase 1: 設計

このUnitはプロンプトファイルの修正のみのため、設計フェーズは簡略化。
技術的考慮事項に記載されている内容に基づき実装。

### Phase 2: 実装

1. `prompts/package/prompts/common/review-flow.md` を読み込み
2. 「AIレビューフロー」セクション（ステップ4）の反復レビュー処理に完了メッセージ出力指示を追加
3. 指摘0件時の分岐処理に「【AIレビュー完了】指摘0件」メッセージを追加
4. テスト: プロンプトの文法・論理フローを確認
