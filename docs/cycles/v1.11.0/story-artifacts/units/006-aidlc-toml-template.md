# Unit: aidlc.tomlテンプレート化

## 概要
新規セットアップ時にaidlc.tomlをテンプレートから生成できるようにし、設定項目の一元管理と新規セットアップの簡素化を実現する。

## 含まれるユーザーストーリー
- C-1: aidlc.tomlテンプレート化 (#90)

## 責務
- aidlc.toml.templateファイルの作成
- setup-prompt.mdの新規セットアップ時にテンプレートから生成するよう修正
- 既存プロジェクトのアップグレード時は従来通りマイグレーションで差分追加

## 境界
- setup-prompt.mdの大規模リファクタリングは行わない
- 既存のマイグレーションロジックは維持
- aidlc.tomlの設定項目自体は変更しない

## 技術的考慮事項
- 作成ファイル: `prompts/package/templates/aidlc.toml.template`
- 修正ファイル: `prompts/setup-prompt.md`
- テンプレートには全設定項目（`[project]`, `[rules]`, `[backlog]` 等）とコメントを含める
- 後方互換性:
  - 新規セットアップ: テンプレートから `docs/aidlc.toml` を生成
  - アップグレード: マイグレーションセクションで新規設定項目を差分追加（既存設定値は保持）
