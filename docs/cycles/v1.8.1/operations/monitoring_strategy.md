# 監視・ロギング戦略 - v1.8.1

## 概要

このプロジェクトはドキュメント・テンプレートプロジェクト（AI-DLC Starter Kit）であり、従来のアプリケーション監視は不要です。

## プロジェクト特性

- **プロジェクトタイプ**: ドキュメント・テンプレート
- **ホスティング**: GitHub リポジトリ
- **実行環境**: なし（ユーザーがクローンして使用）

## 監視対象

### GitHub リポジトリ

| 項目 | 監視方法 | 対応 |
|------|----------|------|
| Issue/PR | 手動確認 | 定期的にチェック |
| Star/Fork | GitHub Insights | 利用状況の把握 |
| トラフィック | GitHub Insights | 利用状況の把握 |

### 自動化された品質チェック

| 項目 | 方法 | 対応 |
|------|------|------|
| Markdownlint | PR Check (GitHub Actions) | PRマージ前に自動チェック |

## 従来の監視が不要な理由

- サーバー/コンテナが存在しない
- データベースが存在しない
- API エンドポイントが存在しない
- ユーザーがローカル環境で使用するため、集中管理が不要

## 将来検討事項

- **GitHub Insightsの定期レビュー**: 利用状況の把握
- **ユーザーフィードバック収集**: Issueやフィードバックテンプレート経由

## v1.8.1での変更

**変更なし** - ドキュメントプロジェクトのため監視設定は不要
