# ユーザーストーリー（簡易版）

## 対象読者
AI活用に興味がある開発者

## ストーリー一覧

### US-1: AI-DLCの概念を理解したい
- 読者として、AI-DLCとは何かを理解したい
- 従来の開発手法との違いを知りたい

### US-2: AI-DLCスターターキットの実践方法を知りたい
- 読者として、AI-DLCスターターキットの使い方を知りたい
- 具体的なワークフローを理解したい

### US-3: 一人AI-DLCの実体験を知りたい
- 読者として、著者の1ヶ月の一人AI-DLC体験を通じて実践イメージを掴みたい
- 一人開発での適用における工夫や課題を知りたい

### US-4: AI-DLCの本質と一人実践の差異を理解したい
- 読者として、本来のAI-DLC（チーム開発想定）と一人実践の違いを知りたい
- 一人でも活用できる部分、限界がある部分を理解したい
