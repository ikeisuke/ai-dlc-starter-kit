# Unit: ユーザー共通設定

## 概要
~/.aidlc/config.tomlによるユーザー共通設定機能を実装する。複数プロジェクトで共通の設定を一元管理できるようにする。

## 含まれるユーザーストーリー
- ストーリー1.2: ユーザー共通設定

## 責務
- ~/.aidlc/config.tomlファイルの読み込みロジック追加
- 3階層設定マージ（ホーム < プロジェクト < .local）
- セットアップ時の~/.aidlc/作成案内

## 境界
- プロジェクト個人設定（.local）の基本マージロジックはUnit 001で実装済み
- 本Unitは3階層統合のみ

## 依存関係

### 依存する Unit
- Unit 001: プロジェクト個人設定（依存理由: マージロジックを共有・拡張）
- Unit 003: Setup/Inception統合（依存理由: セットアップ案内は統合後プロンプトに実装）

### 外部依存
- 環境変数HOME

## 非機能要件（NFR）
- **パフォーマンス**: 設定読み込みに追加の遅延が発生しないこと
- **セキュリティ**: ホームディレクトリのパーミッションに依存
- **スケーラビリティ**: N/A
- **可用性**: ~/.aidlc/がなくても正常動作すること

## 技術的考慮事項
- 環境変数HOMEを使用してホームディレクトリを特定
- Windows対応は考慮外（必要に応じて将来対応）
- セットアップ時にテンプレート作成オプションを提供
