# Unit: Codex resume機能活用

## 概要
AIレビュー時にCodex Skillのresume機能を活用し、同一Unit内でコンテキストを継続できるようにする。

## 含まれるユーザーストーリー
- ストーリー3.1: Codex resume機能活用

## 責務
- AIレビューフローでのセッションID保持
- resume機能を使った継続レビューの実装
- フォールバック処理（resume失敗時の新規セッション開始）

## 境界
- Codex CLI自体の機能変更は対象外
- 他のAIレビューツール（Claude等）への適用は対象外

## 依存関係

### 依存する Unit
- なし

### 外部依存
- Codex CLI（resume機能）

## 非機能要件（NFR）
- **パフォーマンス**: resume利用により初回コンテキスト読み込みを省略
- **セキュリティ**: セッションIDは一時的な値のため、永続化不要
- **スケーラビリティ**: N/A
- **可用性**: resume失敗時は新規セッションにフォールバック

## 技術的考慮事項
- セッションIDの保持方法: Codex出力から抽出し、次回呼び出し時に使用
- Unit切り替え時のセッションリセットロジック
- review-flow.mdの修正が必要

## 実装優先度
Medium

## 見積もり
小規模（review-flow.md修正 + Codex Skill呼び出し方法の変更）
