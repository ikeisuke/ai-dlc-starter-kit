# Unit: プロジェクト個人設定

## 概要
aidlc.toml.localファイルによるプロジェクト個人設定機能を実装する。チーム共有設定と個人の好みを分離し、設定コンフリクトを防止する。

## 含まれるユーザーストーリー
- ストーリー1.1: プロジェクト個人設定

## 責務
- docs/aidlc.toml.localファイルの読み込みロジック追加
- aidlc.tomlと.localのマージ処理（キー単位優先、配列置換、ネスト再帰マージ）
- .gitignoreへの.localファイル追加

## 境界
- ホームディレクトリ設定（~/.aidlc/）はUnit 002で対応
- 3階層マージはUnit 002で統合

- なし

- なし

## 非機能要件（NFR）
- **パフォーマンス**: 設定読み込みに追加の遅延が発生しないこと
- **セキュリティ**: .localファイルに機密情報を含む可能性があるため、gitignoreで保護
- **スケーラビリティ**: N/A
- **可用性**: .localファイルがなくても正常動作すること

## 技術的考慮事項
- プロンプト内のTOML読み込み箇所を特定し、.localファイルの存在確認ロジックを追加
- マージロジックはUnit 002でも再利用可能な形で設計
