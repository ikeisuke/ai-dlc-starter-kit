# AIレビュー必須設定が機能しないことがある

- **発見日**: 2025-12-27
- **発見フェーズ**: Setup
- **発見サイクル**: v1.5.3
- **優先度**: 高

## 概要

AIレビューを必須に設定しているにもかかわらず、レビューがスキップされることがある。原因不明。

## 詳細

**現象**:
- aidlc.toml等でAIレビューを必須に設定
- 実際にはレビューが実行されないケースがある

**考えられる原因**:
- 設定の読み込みタイミングの問題
- 条件分岐のロジックエラー
- プロンプト内でのレビュー呼び出し指示が弱い

## 対応案

1. AIレビューが必須かどうかを判定するロジックを調査
2. 各フェーズのプロンプトでレビュー呼び出し部分を確認
3. 必須設定時の強制実行メカニズムを設計
