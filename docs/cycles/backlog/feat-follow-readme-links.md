# README.md読み込み時にリンクを辿る

- **発見日**: 2025-12-19
- **発見フェーズ**: Setup
- **発見サイクル**: v1.4.1
- **優先度**: 中

## 概要

README.mdを読み込む際に、必要に応じてリンク先のドキュメントも辿って読み込む。

## 詳細

README.mdにはプロジェクトの概要だけでなく、詳細ドキュメントへのリンクが含まれていることが多い。README.mdだけを読んでもプロジェクトの全体像を把握できない場合がある。

## 対応案

- README.md読み込み時に、関連するリンク（特にプロジェクト内部のドキュメントリンク）を検出
- コンテキスト理解に必要そうなリンクは自動的に辿って読み込む
- 関連プロンプトにこのルールを追加
