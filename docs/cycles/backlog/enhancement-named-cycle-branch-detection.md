# Inception時のブランチ名からの名前付きサイクル自動検出

- **発見日**: 2026-03-10
- **発見フェーズ**: Construction
- **発見サイクル**: v1.20.0
- **優先度**: 中

## 概要

Inception Phase実行時に既に名前付きサイクルブランチ（`cycle/waf/v1.0.0` 等）にいる場合、ブランチ名から `cycle_name` を自動検出して名前入力をスキップする。

## 詳細

現状、`mode=named` 時は常に名前入力を求める。しかし既にサイクルブランチ上にいる場合、ブランチ名から `cycle_name` を抽出できる（`suggest-version.sh` は既にこの抽出機能を持つ）。

例:
- `cycle/waf/v1.0.0` にいる → `cycle_name=waf` を自動設定、名前入力スキップ
- `main` にいる → 従来通り名前入力を求める

## 対応案

`inception.md` のステップ5.5で、モード分岐の前にブランチ名チェックを追加:
1. `git branch --show-current` で現在のブランチ取得
2. `cycle/[name]/v*` パターンにマッチする場合、`cycle_name` を抽出して自動設定
3. 自動設定した場合は名前入力をスキップし、バージョン提案に進む
