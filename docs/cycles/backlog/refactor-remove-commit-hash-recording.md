# コミットハッシュのファイル記録を廃止

- **発見日**: 2025-12-19
- **発見フェーズ**: Setup
- **発見サイクル**: v1.4.1
- **優先度**: 中

## 概要

コミットハッシュをファイルに記録する機能を廃止する。

## 詳細

現在、コミットハッシュをファイルに記録する仕組みがあるが、以下の問題がある：

- コミットハッシュを記録するためだけの追加コミットが必要になる
- 「コミット → ハッシュ取得 → ファイル更新 → 再コミット」という無駄なワークフローが発生
- 実質的な価値に対してオーバーヘッドが大きい

## 対応案

- コミットハッシュの記録機能を削除
- 関連するプロンプト・テンプレートから該当箇所を削除
- 必要であれば `git log` で履歴を参照する運用に変更
