# AI MCP・ユーザー入力を検証して自己判断を提示

- **発見日**: 2025-12-20
- **発見フェーズ**: サイクル開始時
- **発見サイクル**: -
- **優先度**: 高

## 概要

AI MCP呼び出しやユーザー入力を受け取る際、信頼するのではなく間違いがあることを前提に、AIの判断を明示的に提示する。

## 詳細

外部からの入力（AI MCPの応答、ユーザーの回答など）をそのまま信頼して処理を進めるのではなく、AIが自身の判断を提示し、ユーザーに確認を求める振る舞いにする。

## 対応案

- 各フェーズプロンプトに「外部入力の検証ルール」を追加
- AI MCPからの応答に対しても批判的に評価し、自己判断を併記
- ユーザー入力に曖昧さがある場合は解釈を明示して確認
