# 名前付きサイクルの既存名一覧選択

- **発見日**: 2026-03-10
- **発見フェーズ**: Construction
- **発見サイクル**: v1.20.0
- **優先度**: 中

## 概要

名前付きサイクルモード有効時、Inception Phaseで既存の名前付きサイクル名を一覧表示し、選択肢として提示する。新規名の入力も可能にする。

## 詳細

現状、`mode=named` 時は自由入力のみ。過去に使用したサイクル名（`waf`, `auth-system` 等）を再利用するケースが多いため、一覧から選べると便利。

例:
```text
名前付きサイクルの名前を選択してください：
1. waf（既存: v1.0.0, v1.1.0）
2. auth-system（既存: v1.0.0）
3. 新しい名前を入力する
```

既存名を選択した場合はバージョン提案に直結し、新規入力の場合は従来の名前入力・バリデーションフローに進む。

## 対応案

`inception.md` のステップ5.5（`mode=named` ブロック）を拡張:
1. `suggest-version.sh` の `all_cycles` から `[name]/v*` パターンを抽出
2. 名前部分をユニーク化して一覧表示
3. AskUserQuestion で選択肢を提示（既存名 + 「新しい名前を入力」）
4. 既存名選択時は `cycle_name` を設定してバージョン提案へ
5. 新規入力選択時は従来の名前入力フローへ
