# Unit: レビューサマリ指摘一覧詳細化

## 概要
レビューサマリファイルの指摘一覧フォーマットを拡張し、問題の具体的なコンテキストと対応内容が後から把握できるようにする。

## 含まれるユーザーストーリー
- ストーリー 5: レビューサマリの指摘一覧詳細化（#247）

## 責務
- `review-flow.md` の指摘一覧テーブルフォーマット定義の拡張（「内容」列・「対応」列の詳細化）
- `review_summary_template.md` テンプレートへの記述例追加
- 「内容」列に対象ファイル・問題の具体的コンテキストを記載するガイダンス
- 「対応」列に修正内容の具体的な変更点を記載するパターン定義

## 境界
- レビュー実行ロジック自体は変更しない
- レビューサマリファイルの基本構造（Set構造等）は変更しない
