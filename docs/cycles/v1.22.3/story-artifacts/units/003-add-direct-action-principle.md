# Unit: 直接実行優先原則の追加

## 概要
`prompts/package/prompts/common/rules.md` に「直接実行優先原則」セクションを追加し、AIエージェントが直接実行を優先するガイドラインを明文化する。

## 含まれるユーザーストーリー
- ストーリー 3: 直接実行優先原則の追加

## 責務
- `prompts/package/prompts/common/rules.md` に「直接実行優先原則」セクションを追加
- 3つの原則（直接実行優先、バックログ後回し禁止、最小複雑性）を記載
- 既存ルール（Overconfidence Prevention原則、改善提案のバックログ登録ルール）との整合性を明文化

## 境界
- 既存ルールの内容変更は行わない（新規セクションの追加のみ）
- 他のプロンプトファイルの変更は対象外

## 技術的考慮事項
- 「改善提案のバックログ登録ルール」はユーザーから依頼されていない改善提案時の規則であり、直接実行優先原則はユーザーから依頼されたタスクの実行方法に関する規則。この区別を明確に記載する
- 「Overconfidence Prevention原則」との関係: 要件が明確な場合は直接実行優先、要件が曖昧な場合は従来通り質問する
