# PRFAQ - AI-DLC Starter Kit v1.22.2

## プレスリリース

### AI-DLC Starter Kit v1.22.2 - セットアップ・デプロイの信頼性向上

AI-DLCスターターキット v1.22.2 をリリースしました。このパッチリリースでは、aidlc-setupスキルのスクリプト不在エラー修正、lib/ディレクトリのデプロイ対応、アップグレード用ブランチ命名の改善、セットアップ時の許可パターン自動設定、CLAUDE.mdへのフェーズ手順明文化を実施しました。

これにより、AI-DLCの初回導入・バージョンアップ時のエラーが解消され、よりスムーズな開発体験を提供します。

## FAQ

### Q1: v1.22.1からアップグレードする場合、何が変わりますか？
A: aidlc-setupスキルのエラーが修正され、正常にアップグレードが完了するようになります。また、セットアップ時にスクリプト実行の許可パターンが自動設定されるため、承認プロンプトの頻度が減少します。

### Q2: アップグレード用ブランチ名はどう変わりますか？
A: `cycle/vX.X.X` から `upgrade/vX.X.X` に変更されます。これにより、開発サイクルのブランチとアップグレード用ブランチが明確に区別できます。

### Q3: 既存プロジェクトへの影響はありますか？
A: 後方互換性を維持しています。既存の設定やワークフローに影響はありません。
