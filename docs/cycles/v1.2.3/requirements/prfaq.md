# PRFAQ: AI-DLC Starter Kit v1.2.3

## プレスリリース

### AI-DLC Starter Kit v1.2.3 リリース - AI動作の安定性と開発者体験を向上

**2025年12月** - AI-DLC Starter Kit v1.2.3 をリリースしました。本バージョンでは、運用中に発見された6つの問題を修正し、AIの動作安定性と開発者体験を向上させました。

#### 主な改善点

1. **Lite版プロンプトのパス解決を安定化** - ファイルパスを明確化し、AIが迷わず正しいファイルにアクセスできるようになりました。

2. **フェーズ遷移のガードレールを強化** - 各フェーズで禁止されるアクションを明示し、AIがフェーズを無視して先走り実装する問題を防止します。

3. **バージョン管理の改善** - aidlc.tomlにstarter_kit_versionフィールドを追加し、バージョン比較が正しく動作するようになりました。

4. **移行時の安全性向上** - 旧バージョンからの移行時に削除されるファイルを事前確認できるようになり、意図しないファイル削除を防止します。

5. **履歴記録の正確性向上** - 日時記録時に必ず現在時刻を取得するルールを必須化し、履歴の日時が正確になりました。

6. **フェーズ責務の明確化** - Inception PhaseからConstruction用ファイル作成を削除し、各フェーズの責務を明確化しました。

#### アップグレード方法

既存プロジェクトは、セットアッププロンプトを実行することでアップグレードできます。

## FAQ

### Q1: v1.2.2からのアップグレードは必要ですか？
A1: 推奨します。特にLite版を使用している場合や、フェーズ遷移で問題が発生している場合は、本バージョンで改善されます。

### Q2: 破壊的変更はありますか？
A2: ありません。パッチバージョンのため、既存の設定やワークフローに影響はありません。

### Q3: Inception Phaseのステップ数が変わりましたか？
A3: はい。ステップ6（Construction用進捗管理ファイル作成）を削除し、5ステップ構成になりました。Construction Phaseで自身がファイルを作成します。

### Q4: 旧バージョン（v1.1以前）からの移行は安全ですか？
A4: はい。本バージョンでは移行時に削除されるファイルを事前に確認できるため、より安全に移行できます。
