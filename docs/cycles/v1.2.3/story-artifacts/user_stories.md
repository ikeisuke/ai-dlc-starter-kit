## ストーリー一覧

### US-1: Lite版パス解決の安定化

**ストーリー**:
As a AI-DLCを使用する開発者
I want Lite版プロンプトでファイルパスが明確に示される
So that AIが迷わず正しいファイルにアクセスできる

**受け入れ基準**:
- [ ] Lite版プロンプトに具体的なパス例が記載されている
- [ ] サイクルディレクトリ構造の説明が追加されている
- [ ] AIがファイルを探す際に「見つからない」エラーが減少する

---

### US-2: フェーズ遷移ガードレールの強化

**ストーリー**:
As a AI-DLCを使用する開発者
I want 各フェーズで禁止されるアクションが明示される
So that AIがフェーズを無視して先走り実装しない

**受け入れ基準**:
- [ ] setup-cycle.mdの完了メッセージに「Inception Phaseで計画を立ててから実装」が明示されている
- [ ] inception.mdに「このフェーズでは実装コードを書かない」が記載されている
- [ ] 各プロンプトにフェーズごとの許可/禁止アクションが明確化されている
