## デプロイ手順

### 1. 最終確認
```bash
# 変更ファイルの確認
git status

# 差分確認
git diff main
```

### 2. mainブランチへマージ
```bash
# PRを作成してマージ、またはローカルでマージ
git checkout main
git merge cycle/v1.2.1
```

### 3. タグ作成
```bash
git tag v1.2.1
git push origin v1.2.1
```

### 4. 動作確認
- [ ] 別ディレクトリでクローンしてセットアップが動作する
- [ ] 各フェーズプロンプトが読み込める

## ロールバック手順

問題が発生した場合：

```bash

git tag -d v1.2.1

# または前のバージョンに戻す
git checkout v1.2.0
```
