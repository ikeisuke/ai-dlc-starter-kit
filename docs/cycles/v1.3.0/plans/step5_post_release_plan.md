# ステップ5: リリース後の運用 計画

## 概要

v1.3.0 リリース後の運用方針を記録する。このプロジェクトはドキュメント・テンプレートプロジェクトのため、サーバー運用ではなく、リポジトリの運用方針を記録する。

1. テンプレートをこのプロジェクト向けにカスタマイズ
