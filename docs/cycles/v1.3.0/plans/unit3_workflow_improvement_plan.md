### 5. PRマージ後の手順【重要】

PRがマージされたら、次サイクル開始前に以下を実行：

1. **mainブランチに移動**:
   ```bash
   git checkout main
   ```

2. **最新の変更を取得**:
   ```bash
   git pull origin main
   ```

3. **マージ済みブランチの削除（任意）**:
   ```bash
   git branch -d cycle/vX.X.X
   ```

**注意**: この手順を実行してから次サイクルのセットアップを開始してください。
