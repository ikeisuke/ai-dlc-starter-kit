# Unit: バージョン管理改善

## 概要

Operations Phaseでのバージョン更新が正しく動作するよう修正し、初回セットアップ時のバージョン提案も改善する。

## 含まれるユーザーストーリー

- US-2: Operations Phaseのバージョン更新修正
- US-3: 初回セットアップ時のバージョン提案改善

## 責務

- version.txtとaidlc.tomlのstarter_kit_versionの整合性確保

- バージョン更新手順のドキュメント化
- 初回セットアップ時のプロジェクト既存バージョン調査手順の追加

## 境界
