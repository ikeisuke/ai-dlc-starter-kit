# Unit: ワークフロー改善

## 概要

AI-DLCの各種ワークフローを改善し、より効率的で漏れのない開発フローを実現する。

## 含まれるユーザーストーリー

- US-4: Unit作業中の気づき記録フロー定義
- US-5: コンテキストリセットのタイミング見直し
- US-6: Unit開始前のバックログ確認ステップ化
- US-7: PRマージ後の手順明確化

## 責務

- 気づき記録フローの定義とプロンプトへの組み込み
- コンテキストリセット提案タイミングの修正（Unit完了時→Unit開始時）
- Construction Phase開始時のバックログ確認ステップ追加
- PRマージ後の手順ドキュメント化

## 境界

- プロンプトの大幅な構造変更は含まない
- 新機能追加ではなく、既存フローの改善のみ
