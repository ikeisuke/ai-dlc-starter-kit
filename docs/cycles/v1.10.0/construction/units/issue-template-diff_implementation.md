# 実装記録: Issueテンプレート差分確認機能

## 概要

Issueテンプレートのコピー前に差分を確認し、変更がある場合のみユーザーに確認を求める機能を追加。

## 実装内容

### 変更ファイル

| ファイル | 変更内容 |
|----------|----------|
| `prompts/setup-prompt.md` | セクション8.2.5の差分確認ロジック追加 |

### 主な変更点

1. **差分確認ロジック追加**
   - `diff -q` コマンドで各ファイルの差分を確認
   - 差分のあるファイルと新規ファイルを区別して管理

2. **スキップ処理**
   - 差分も新規ファイルもない場合は「スキップします」メッセージを表示

3. **ユーザー確認フロー**
   - 差分がある場合の選択肢:
     1. 上書きする（推奨）
     2. スキップする
     3. 差分を確認してから決める

4. **対象ファイル追加**
   - `feedback.yml` を対象ファイルに追加

## テスト結果

プロンプト修正のみのため、コードテストは不要。
