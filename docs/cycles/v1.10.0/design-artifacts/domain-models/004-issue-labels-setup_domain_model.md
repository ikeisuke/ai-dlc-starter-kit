# ドメインモデル: Issue用基本ラベル移動

## 概要

`init-labels.sh` の呼び出し位置を変更する。

## ドメイン概念

### ラベル初期化タイミング

| 現状 | 変更後 |
|------|--------|
| サイクル開始時（setup.md） | 初期セットアップ時（setup-prompt.md） |

### 理由

- 基本ラベルは1回作成すれば十分
- 毎回のサイクル開始時に確認する必要がない
- `init-labels.sh` は冪等性があるため、2回目以降は既存ラベルをスキップ

## 影響範囲

- 初期セットアップ時: ラベル作成処理が追加される
- サイクル開始時: ラベル確認ステップがなくなる（高速化）
