# ドメインモデル: Issueテンプレート差分確認

## 概要

Issueテンプレートコピー時に差分を確認し、変更がある場合のみユーザー確認を行う機能。

## エンティティ

### IssueTemplateDiff

Issueテンプレートの差分状態を表す。

| 属性 | 型 | 説明 |
|------|-----|------|
| sourceDir | Path | スターターキット側のテンプレートディレクトリ |
| targetDir | Path | プロジェクト側のテンプレートディレクトリ |
| diffFiles | List[String] | 差分のあるファイル一覧 |
| hasDiff | Boolean | 差分の有無 |

## 値オブジェクト

### DiffResult

差分確認の結果。

| 値 | 説明 |
|----|------|
| NO_TARGET | ターゲットディレクトリが存在しない（新規コピー） |
| NO_DIFF | 差分なし（スキップ） |
| HAS_DIFF | 差分あり（ユーザー確認が必要） |

### UserChoice

ユーザーの選択。

| 値 | 説明 |
|----|------|
| OVERWRITE | 上書きする |
| SKIP | スキップする |
| SHOW_DIFF | 差分を確認してから決める |

## 処理フロー

```text
[開始]
   │
   ▼
[ターゲットディレクトリ存在確認]
   │
   ├─ 存在しない → [新規コピー実行] → [終了]
   │
   ▼ 存在する
[diff -q で差分確認]
   │
   ├─ 差分なし → [スキップメッセージ表示] → [終了]
   │
   ▼ 差分あり
[差分ファイル一覧表示]
   │
   ▼
[ユーザー選択]
   │
   ├─ 1. 上書き → [コピー実行] → [終了]
   ├─ 2. スキップ → [終了]
   └─ 3. 差分確認 → [diff 詳細表示] → [ユーザー選択に戻る]
```
