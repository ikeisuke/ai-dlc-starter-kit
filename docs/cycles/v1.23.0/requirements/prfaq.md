# PRFAQ

## プレスリリース

AI-DLCスターターキット v1.23.0 では、全フェーズプロンプトのリファクタリングを実施しました。枝番ステップの連番化、冗長記述の削減、未使用スクリプトの削除により、プロンプトの可読性と保守性が大幅に向上しています。また、README.mdを刷新し、新規ユーザーが迷わず使い始められるクイックスタートを提供します。

## FAQ

### Q: フローの動作は変わりますか？
A: いいえ。ステップ番号の振り直しと表現の簡潔化のみで、フローの機能自体は変更しません。

### Q: 何が改善されますか？
A: ステップ番号が整数連番に統一され、ハイフン記法やアルファベット記法の混在が解消されます。冗長な記述も削減されるため、AIのコンテキスト消費が減り応答精度が向上します。

### Q: README.mdはどう変わりますか？
A: バージョンリリースノート（約1000行）をCHANGELOG.mdに移動し、クイックスタートを中心とした簡潔な構成（500行以下）にリライトします。
