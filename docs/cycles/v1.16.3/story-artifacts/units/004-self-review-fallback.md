# Unit: セルフレビューフォールバック

## 概要
外部AIレビューツール（Skills）が利用できない場合に、実行中のエージェント自身がセルフレビューを実施するフォールバック機能を追加する。

## 含まれるユーザーストーリー
- ストーリー 6: セルフレビューフォールバック (#216)

## 責務
- review-flow.md のステップ3（利用可否チェック）にセルフレビューパスを追加
- review-flow.md のステップ6（AIレビュー不可時）にセルフレビュー選択肢を追加
- セルフレビューの実行手順・出力フォーマットを定義
- 履歴記録にセルフレビューであることを明記する形式を定義

## 境界
- 外部レビューツール（Codex, Claude CLI, Gemini）自体のインストール・設定の自動化は行わない
- レビュースキル（reviewing-code, reviewing-architecture 等）のSKILL.md自体の変更は行わない

## 技術的考慮事項
- セルフレビューは外部ツールに比べて品質が劣る可能性があるため、履歴に明記
- mode=required 時はセルフレビューが最優先、mode=recommend 時はセルフレビューを推奨
- セルフレビューでも通常と同じレビュー観点（指摘番号・重要度付き）を適用
- 変更対象は prompts/package/ 配下
