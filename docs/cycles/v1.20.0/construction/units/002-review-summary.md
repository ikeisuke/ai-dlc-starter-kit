## 技術的知見

- Bashの`$()`コマンド置換はサブシェルで実行されるため、内部で設定した変数は親シェルに伝播しない
- git refではTAB文字が禁止されているため、ブランチ名由来の値のセパレータとして安全
- `|`文字はgit refで許容されるため、セパレータとして不適切
