# Unit 7: プロンプト分割・短縮化

## 概要
フェーズプロンプトを小さいファイルに分割し、AIが確実に内容を読むようにする

## 含まれるユーザーストーリー
- ストーリー 5.1: プロンプトの分割・短縮化

## 責務
- 各フェーズプロンプトの分割
- ステップごとの必要最小限の情報整理
- 読み込み確認機構の検討

## 境界
- プロンプト分割のみ
- AIの動作変更は行わない

## 依存関係

### 依存するUnit
- Unit 4: バージョン管理（分割後のファイルにもバージョン埋め込み）
- Unit 5: セットアップ分離（分割ファイルの生成に対応）
- Unit 6: プロンプト生成方式改善（テンプレート分割と連携）

### 外部依存
- なし

## 非機能要件（NFR）
- **パフォーマンス**: AIの応答品質向上
- **セキュリティ**: 該当なし
- **スケーラビリティ**: 該当なし
- **可用性**: 該当なし

## 技術的考慮事項
- 分割案:
  ```
  docs/aidlc/prompts/
  ├── inception/
  │   ├── init.md       # 初期化・確認
  │   ├── flow.md       # フロー定義
  │   └── complete.md   # 完了処理
  ├── construction/
  │   ├── init.md
  │   ├── flow.md
  │   └── complete.md
  └── operations/
      ├── init.md
      ├── flow.md
      └── complete.md
  ```
- または単一ファイルを短縮（冗長な説明を削減）
- Claude Code以外（Kiro等）でも安定動作を確認

High
