# Unit 5 実装計画: アップグレード時の変更要約表示

## 概要

アップグレード時にrsync実行後、更新されたファイルと主な変更点をユーザーに表示する機能を追加します。

  - セクション7.2.2（テンプレートの同期）の後に変更要約表示を追加

---

## 変更内容

### セクション7.2.1と7.2.2の修正

rsync実行後に、更新されたファイル一覧を表示するセクションを追加：

```markdown
#### 7.2.1.1 変更内容の要約表示

rsync実行後、以下のような要約を表示：

```
更新されたファイル:
- [更新されたファイル1]
- [更新されたファイル2]
...

主な変更点:
- 各フェーズプロンプトの最新版に更新
- テンプレートの最新版に更新
```
```

### 実装方針

rsyncの出力を解析して更新されたファイルを抽出し、ユーザーに分かりやすい形式で表示します。
