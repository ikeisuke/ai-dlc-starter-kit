# Unit: コミット処理統合

## 概要

分散しているコミット関連のロジックを `prompts/package/prompts/common/` に集約し、各フェーズプロンプトからの参照を統一する。squash機能もこの統合フローに組み込む。

## 責務

- 各フェーズプロンプトからのコミット処理参照の統一
- squash機能の統合フローへの組み込み

## 境界

- squashスクリプト自体の実装はUnit 001で完了済み
- プロンプトへのsquash組み込みの基本部分はUnit 002で完了済み
- プロンプト全体のリファクタリングはUnit 005で実施
- Skillファイル作成は次サイクル以降

## 依存関係

### 依存する Unit

- Unit 001: squashスクリプト作成（依存理由: squash機能をコミットフローに統合するため）
- Unit 002: プロンプトへのsquash組み込み（依存理由: 基本的なsquash組み込みが完了している前提で統合するため）
- Unit 003: プロンプト構造分析・方針策定（依存理由: 方針に基づいてコミット処理の集約先を決定するため）
