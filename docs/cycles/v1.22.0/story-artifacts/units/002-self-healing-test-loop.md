# Unit: Self-Healingテストループ

## 概要
Construction Phase Step 6でテスト失敗時に自動修正ループ（最大3回）を導入し、ユーザー介入を削減する

## 含まれるユーザーストーリー
- ストーリー 2

## 責務
construction.md Step 6への自動修正ループ追加、attempt番号と失敗要因の出力仕様定義、3回失敗時のユーザー判断フロー。非回復系エラー（認証失効、ツール制約等）は即時フォールバックとしリトライ対象外とする

## 境界
レビューツール実行不可の対応はUnit 003の範囲。リトライ回数の設定化（#322）は今回のスコープ外。非回復系エラーの分類はUnit 003の制約情報を参照する
