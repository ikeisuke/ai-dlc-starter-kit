# Unit: 複合コマンド削減

## 概要
プロンプト内の複合コマンド（&&, ||）を削減し、事前確認をフェーズ/Unit開始時に集約する。

## 含まれるユーザーストーリー
- ストーリー 5: 複合コマンド削減

## 責務
- 事前確認（gh auth status, command -v等）をフェーズ開始時に集約
- 複合コマンドを可能な限り単一コマンドに分解
- 対象ファイル: setup.md, inception.md, construction.md, operations.md
