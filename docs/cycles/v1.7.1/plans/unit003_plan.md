# Unit 003 実装計画: AskUserQuestion推奨オプション順序ルール

## 概要

CLAUDE.mdにAskUserQuestionの選択肢順序ルールを追記する。

## 対象ファイル

- `prompts/package/prompts/CLAUDE.md`（メタ開発のため）

## Phase 1: 設計

### ドメインモデル設計

この変更は**ドキュメント追記**であり、新規のドメインモデルは不要。

### 論理設計

既存の「AskUserQuestion機能の活用」セクションに順序ルールを追記：

- 推奨オプションを一番上（最初の選択肢）に配置する
- 推奨オプションには「(推奨)」または「(Recommended)」を末尾に付加する

## Phase 2: 実装

### 作業内容

1. `prompts/package/prompts/CLAUDE.md` の「AskUserQuestion機能の活用」セクションを編集
2. 選択肢順序ルールを追記

### 変更内容（予定）

```markdown
### AskUserQuestion機能の活用

選択肢が明確な場合はAskUserQuestion機能を使用してください：
- 技術的な選択肢がある場合（例: 「どのライブラリを使用しますか？」）
- Yes/No の判断が必要な場合（例: 「この設計で進めてよいですか？」）
- 複数の対応案から選択する場合

**選択肢の順序ルール**:
- 推奨オプションがある場合は、一番上（最初の選択肢）に配置する
- 推奨オプションには「(推奨)」または「(Recommended)」をラベル末尾に付加する
```

## 完了条件

- [ ] CLAUDE.mdに順序ルールが追記されている
- [ ] ドキュメントのフォーマットが整っている
