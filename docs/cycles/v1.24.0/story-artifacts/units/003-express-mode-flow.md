# Unit: エクスプレスモードフロー実装

## 概要
inception.mdとconstruction.mdにエクスプレスモードのフロー分岐（Inception+Construction統合パス）を実装する。

## 含まれるユーザーストーリー
- ストーリー 1b: Inception+Construction統合フロー（#359）

## 責務
- inception.mdにエクスプレスモード判定分岐を追加（Depth Level読み込み後）
- inception.mdのエクスプレスモード時フロー（簡略Intent→簡略ストーリー→実装への自動遷移）
- construction.mdにエクスプレスモード用の簡略フロー（設計省略→直接実装）を追加
- エクスプレスモード完了時のコミットフロー記述
- フォールバック時の通知メッセージ適用（文言の正本はrules.md（Unit 002）を参照）
- エクスプレスモード完了失敗時のエラーメッセージ表示と手動対応案内

## 境界
- エクスプレスモードの仕様定義（適用条件等）はUnit 002で完了済みを前提とする
- Operations Phase統合は対象外
- 新規スクリプトの追加は行わない
