# Unit: Operations Phaseサイズチェック

## 概要

プロンプトファイルのサイズが閾値を超えた場合に警告を表示する機能を追加する。

## 含まれるユーザーストーリー

- ストーリー5: Operations Phaseサイズチェック追加 (#79)

## 責務

- サイズチェックスクリプトの作成
- 閾値設定の実装
- 警告メッセージの表示
- Operations Phase完了時チェックリストへの組み込み

## 境界

- 参照漏れチェックは別Unit（Unit 004）
- 自動修正は行わない（警告のみ）

## 技術的考慮事項

- 閾値はaidlc.tomlで設定可能にする
- 行数またはバイト数で測定
- デフォルト閾値の設定（例: 1000行）
- 出力形式: ファイル名、現在のサイズ、閾値、警告メッセージ
