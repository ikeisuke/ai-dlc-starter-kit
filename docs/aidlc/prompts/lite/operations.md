# Operations Phase プロンプト（Lite版）

## Full版プロンプトの参照【必須】

**まず以下のFull版プロンプトを読み込んでください**:
`docs/aidlc/prompts/operations.md`

Full版の内容を理解した上で、以下の変更点を適用してください。

---

## Lite版での変更点

### 全ステップが任意

Lite版では**Operations Phase自体が任意**です。

**スキップ可能な条件**:
- CI/CDが既に構築済み
- 監視・ロギングが既に設定済み
- 配布プロセスが確立済み
- 軽微な変更でデプロイ手順の変更が不要

### 実施する場合

必要なステップのみを選択して実施：

1. **デプロイ準備**（必要な場合のみ）
   - 環境変数の追加・変更がある場合
   - 依存関係の追加がある場合

2. **リリース後確認**（推奨）
   - 変更が正しく反映されたか確認
   - エラーが発生していないか確認

### 維持するルール

Operations Phase を実施する場合、以下は維持：

- Gitコミット
- 履歴記録（history.md）

---

## Lite版の完了基準

以下のいずれかを満たすこと：

1. **スキップの場合**: 「Operations Phaseをスキップ」を明示的に選択・記録
2. **実施の場合**: 必要な運用作業の完了

---

## サイクル完了

Operations Phase (Lite) が完了（またはスキップ）したら、サイクルは完了です。

### 履歴記録

history.mdにサイクル完了を記録してください。

### 次のサイクル開始

新しいサイクルを開始する場合：

```
以下のファイルを読み込んで、AI-DLC の次サイクル の環境をセットアップしてください：
[setup-prompt.mdのパス]

変数を以下に設定してください：
- CYCLE = vX.X.X（次のサイクル識別子）
- CYCLE_TYPE = Full または Lite
```
