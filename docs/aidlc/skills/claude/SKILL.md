# Claude

Claude Code CLIを使用してコードレビュー・分析を実行するスキル。

## 実行コマンド

```bash
claude -p "<request>"
```

## パラメータ

| パラメータ | 説明 |
|-----------|------|
| `-p` / `--print` | 非対話モードで実行し結果を表示 |
| `-r` / `--resume` | セッションの再開（ID指定、または検索画面を表示） |
| `--session-id <uuid>` | 特定のセッションIDを指定して再開 |
| `"<request>"` | 依頼内容（日本語可） |

## 使用例

### コードレビュー（新規）

```bash
claude -p "このプロジェクトのコードをレビューして、改善点を指摘してください"
```

### コンテキストを引き継いで実行（推奨手順）

意図しないセッションの混同を防ぐため、`-c` (continue) ではなく以下の方法を推奨します。

1. **手動でセッションを選択する場合**
   ユーザーが対話画面で適切なセッションを選択してから、追加の指示を送ります。

   ```bash
   claude -r
   ```

2. **特定のセッションIDを直接指定する場合**

   ```bash
   claude --session-id <uuid> -p "先ほどの指摘箇所の修正を確認してください"
   ```

## 実行手順

1. ユーザーから依頼内容を受け取る
2. 対象プロジェクトのディレクトリに移動
3. 新規か継続かに応じてコマンドを選択
   - 新規: `claude -p "..."`
   - 継続: ユーザーにIDを確認するか、`claude -r` での選択を促す
4. 結果をユーザーに報告
