# Unit: 各フェーズプロンプトのパス参照更新

## 概要
inception.md、construction.md、operations.md のパス参照を新しいディレクトリ構造に対応させる

## 含まれるユーザーストーリー
- ストーリー 13: inception.md のパス参照更新
- ストーリー 14: construction.md のパス参照更新
- ストーリー 15: operations.md のパス参照更新

## 責務
- 各フェーズプロンプト内のパス参照を `{{AIDLC_ROOT}}/templates/` と `{{VERSIONS_ROOT}}/{{VERSION}}/` に更新
- テンプレート参照パスの置換
- 成果物ディレクトリ参照パスの置換
- setup-prompt.md への「セットアッププロンプトパス」記載の確認

## 境界
- プロンプトの内容（フロー、ルール等）の変更は含まない
- パス参照の置換のみ

## 依存関係

### 依存する Unit
- Unit1: setup-prompt.md のリファクタリング（依存理由: 新しい変数定義が必要）

### 外部依存
- なし

## 非機能要件（NFR）
- **パフォーマンス**: 特に制約なし
- **セキュリティ**: 特に制約なし
- **スケーラビリティ**: パス参照が正確であること
- **可用性**: エラーのないパス置換

## 技術的考慮事項
- setup-prompt.md の変数置換ロジックを利用
- セットアップ時に自動置換される仕組みを維持

## 実装優先度
High

## 見積もり
半日
