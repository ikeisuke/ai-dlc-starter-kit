# 追加ルール（プロジェクト固有）

このファイルには、AI-DLC Starter Kit v1.0.0 開発における追加ルールと制約を記載します。

## 実行前の検証ルール

### Codex MCPレビュー（オプション）

プロンプトファイルを実行する前に、Codex MCPでプロンプトの妥当性を確認することを推奨します。

### 指示の妥当性検証

- 各フェーズのプロンプトファイルに記載された手順に従う
- 不明点がある場合は、ユーザーに質問して明確化する

## フェーズ固有のルール

### Inception Phase

- Intent作成時は、v1.0.0-intent.md を参照する
- 新しいディレクトリ構造への移行が主要な要件

### Construction Phase

- 既存の docs/example/ 構造を参考にする
- 後方互換性を考慮する

### Operations Phase

- GitHub Actions によるCI/CD設定
- バージョンタグの作成

## 禁止事項

- 既存のv0.1.0ドキュメントを削除・変更しない
- ユーザーの明示的な承認なしに大規模な変更を実施しない

## コーディング規約

- Markdownファイルは UTF-8 エンコーディング
- 日本語で記述
- 見出しレベルは適切に使用（# から #### まで）
