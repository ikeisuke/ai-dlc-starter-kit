# AI-DLC Starter Kit v1.0.0 リリースノート

## リリース日
2025-11-25

## 概要
AI-DLC Starter Kit v1.0.0 は、プロンプトとテンプレートの共通化により、メンテナンス性とスケーラビリティを大幅に向上させたメジャーアップデートです。

## 主要な変更点

### 1. 新しいディレクトリ構造
プロンプトファイルとバージョン固有成果物を分離し、メンテナンス性を向上：

```
docs/
├── aidlc/                 # バージョン非依存の共通ファイル（v1.0.0以降）
│   ├── prompts/          # 共通プロンプト
│   └── templates/        # 共通テンプレート
│
└── versions/              # バージョン固有の成果物
    └── v1.0.0/
        ├── prompts/       # v0.1.0で生成（このプロジェクト自体の開発記録）
        ├── requirements/
        ├── construction/
        └── operations/
```

**v0.1.0との違い**:
- v0.1.0: `{DOCS_ROOT}/{VERSION}/prompts/` にバージョンごとにプロンプトを生成
- v1.0.0: `docs/aidlc/prompts/` に共通プロンプトを配置（全バージョンで共有）

### 2. プロンプトの共通化
- スターターキット改善時に全バージョンに自動反映
- 各フェーズプロンプト（inception.md, construction.md, operations.md）にcommon.mdの内容を統合
- ユーザーは1ファイル読み込むだけでOK

### 3. バージョン管理機能
- `docs/aidlc/version.txt` でスターターキットのバージョンを記録
- セットアップ時にバージョンを自動記録

### 4. テンプレートの共通化
- `docs/aidlc/templates/` に共通テンプレートを配置
- JIT生成機能を維持しつつ、生成先を共通ディレクトリに変更

## 期待される効果

### 既存ユーザーへの価値
1. **イニシャルコストの激減**: v1.0.0以降、新バージョン開始時のセットアップコストが大幅削減
2. **継続的な改善の恩恵**: スターターキット改善が自動的に全バージョンに反映
3. **一貫性の向上**: すべてのバージョンで同じプロンプトを使用

### メンテナンスへの価値
1. **メンテナンス負荷の軽減**: プロンプト更新が1箇所で完結
2. **スケーラビリティの向上**: 新バージョン追加時の作業量が削減
3. **品質の向上**: 共通化により、バグ修正や改善が全バージョンに即座に反映

## 破壊的変更
なし。v0.1.0で生成された成果物はそのまま動作します。

## マイグレーションガイド

### v0.1.0からv1.0.0への移行

**既存プロジェクトの継続**:
- v0.1.0で生成したプロジェクトはそのまま使用可能
- 変更不要

**新規プロジェクトの開始**:
v1.0.0を使って新しいプロジェクトをセットアップすると、新しいディレクトリ構造が使用されます：

```markdown
以下のファイルを読み込んで、AI-DLC 開発環境をセットアップしてください：
/path/to/ai-dlc-starter-kit/prompts/setup-prompt.md
```

セットアップ後、`docs/aidlc/prompts/inception.md` 等を読み込んで開発を開始します。
