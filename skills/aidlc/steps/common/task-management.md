# タスク管理ガイド

各フェーズ・ワークフロー開始時に、具体的なタスクリストを作成する。
これにより、AIレビュー漏れ・手順スキップ・完了処理忘れを防止する。

## 原則

1. **ワークフロー開始時にタスクを一括作成**: フェーズやUnit開始時に、以下のテンプレートに基づいてタスクを作成する。**タスクリスト未作成のまま作業を開始してはいけない**
2. **ステップ着手時に `in_progress`**: 各タスクに着手したら即座にステータスを更新する
3. **ステップ完了時に `completed`**: 完了したら即座にマークする
4. **スキップ時も `completed`**: depth_level等でスキップする場合も、理由を添えて完了マークする

## Construction Phase: Unit開始時タスクテンプレート

対象Unit決定後（`01-setup.md` ステップ8a）に以下のタスクを作成する。
`depth_level=minimal` でスキップするステップは作成不要。

### Phase 1: 設計

| タスク | 条件 |
|--------|------|
| ドメインモデル設計 | `depth_level` != `minimal` |
| 論理設計 | `depth_level` != `minimal` |
| 設計AIレビュー（review-flow.md） | `depth_level` != `minimal` |
| 設計承認 | `depth_level` != `minimal` |

### Phase 2: 実装

| タスク | 条件 |
|--------|------|
| コード生成 | 常に |
| コードAIレビュー（review-flow.md） | 常に（`review_mode=required` の場合はスキップ不可） |
| テスト生成 | 常に |
| ビルド・テスト実行 | 常に |
| 統合AIレビュー（review-flow.md） | 常に（`review_mode=required` の場合はスキップ不可） |
| 実装承認 | 常に |

### 完了処理

| タスク | 条件 |
|--------|------|
| 完了条件チェック | 常に |
| 設計・実装整合性チェック | 設計ドキュメントが存在する場合 |
| 意思決定記録参照確認 | 常に（記録対象の有無を確認し、対象ありなら追記/なしなら明示報告） |
| AIレビュー実施確認 | 常に |
| Unit定義ファイル状態更新（→完了） | 常に |
| 履歴記録（`/write-history` スキル） | 常に |
| Markdownlint実行 | `markdown_lint=true` の場合 |
| Squash | `squash_enabled=true` の場合 |
| Gitコミット | 常に |
| コンテキストリセット提示 | 常に |

## Inception Phase: タスクテンプレート

### Part 1: セットアップ（プリフライト完了後、`01-setup.md` ステップ1a直後に作成）

| タスク | 条件 |
|--------|------|
| サイクルバージョン決定・ブランチ作成 | 常に |
| サイクルディレクトリ作成 | 常に |
| インセプション準備（Issue・バックログ確認） | 常に |

### Part 2以降（`01-setup.md` ステップ12b、サイクルディレクトリ作成後に作成）

| タスク | 条件 |
|--------|------|
| Intent作成 | 常に |
| Intent AIレビュー | 常に |
| ストーリー・Unit定義 | 常に |
| 意思決定記録確認 | 常に（記録対象の有無を確認し、対象ありなら記録/なしなら明示報告） |
| エクスプレス適格性評価 | `express_enabled=true` の場合 |
| Inception完了処理（履歴・squash・コミット） | 常に |
| コンテキストリセット提示 | 常に |

## Operations Phase: タスクテンプレート

Operations Phase 進捗管理ファイル確認後（`01-setup.md` ステップ7a）に以下のタスクを作成する。

| タスク | 条件 |
|--------|------|
| デプロイ準備（チェックリスト・CI/CD・監視） | 常に |
| バックログクリーンアップ | 常に |
| リリース準備（PR作成・マージ） | 常に |
| ポストマージクリーンアップ | 常に |
| コンテキストリセット提示 | 常に |
