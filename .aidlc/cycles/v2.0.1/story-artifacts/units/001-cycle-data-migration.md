# Unit: サイクルデータ移行

## 概要
`docs/cycles/` 配下の全サイクルデータを `.aidlc/cycles/` に移動し、関連スクリプトのパス参照を更新する。

## 含まれるユーザーストーリー
- ストーリー 1: サイクルデータの統一パス化

## 責務
- `docs/cycles/` → `.aidlc/cycles/` への全サイクルデータ移動（このリポジトリ内）
- `suggest-version.sh` 等のサイクル参照スクリプトのパス更新
- `docs/cycles/` ディレクトリの削除
- 移行フロー文書（`03-migrate.md`）の更新

## 境界
- 利用プロジェクト側の移行は扱わない（#416 移行スキルで対応）
- `.aidlc/cycles/rules.md` の内容変更は行わない
