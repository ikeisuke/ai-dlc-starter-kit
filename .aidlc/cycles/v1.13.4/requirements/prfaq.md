# PRFAQ: AI-DLC Starter Kit v1.13.4

## Press Release（プレスリリース）

**見出し**: AI-DLC Starter Kit v1.13.4 - AIレビュースキルの改善・安定化

**副見出し**: Codex CLIとのスキル連携を強化し、claude-reviewの安定性を向上

**発表日**: 2026-02

**本文**:

AI-DLCスターターキットは、AIを開発プロセスの中心に据えた開発方法論の実践キットです。v1.13.4では、AIレビュースキルの利便性と安定性を向上させるパッチリリースを行いました。

これまでCodex CLIでAI-DLCのレビュースキルを利用するための設定方法が不明確であり、compatibilityフィールドも未記載のためサンドボックス要件がわかりにくい状態でした。また、claude-reviewスキルではレスポンスが返らない場合や、反復レビューで指摘が二転三転する不安定動作が報告されていました。

v1.13.4では、Codex skillsの設定ドキュメントを整備し、compatibilityフィールドでサンドボックス要件を明記しました。さらに、claude-reviewスキルの不安定動作を調査し、対策を実装またはワークアラウンドを明文化しました。

「AIレビューが安定して動作することで、開発サイクルを中断なく回せるようになりました」と利用者は述べています。

今後もAIレビュースキルの品質改善を継続し、より多くのAIツールとの連携を強化していく予定です。

## FAQ（よくある質問）

### Q1: Codex CLIでAI-DLCスキルを使うにはどうすればよいですか？
A: v1.13.4で追加された設定ドキュメントに従い、`~/.codex/skills/` にシンボリックリンクを手動で作成してください。詳細はガイドドキュメントを参照してください。

### Q2: claude-reviewの不安定動作はどのように改善されましたか？
A: レスポンス未返却と指摘の二転三転の原因を調査し、スキル設定の改善またはワークアラウンドを明文化しました。具体的な対策内容はリリースノートを参照してください。

### Q3: compatibilityフィールドとは何ですか？
A: Agent Skills Specificationで定義されたメタデータフィールドで、スキルが必要とするサンドボックス要件（ネットワークアクセス等）をドキュメントとして記載します。現時点ではランタイムでの強制はなく、情報提供の目的です。
