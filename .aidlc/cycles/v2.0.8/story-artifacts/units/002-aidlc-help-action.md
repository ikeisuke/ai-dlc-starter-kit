# Unit: /aidlc help アクション追加

## 概要
`/aidlc help` (`/aidlc h`) コマンドで利用可能なアクション一覧とヘルプ情報を表示する機能を追加する。

## 含まれるユーザーストーリー
- ストーリー 4: /aidlc help アクション追加（#464）

## 責務
- SKILL.md の ARGUMENTSパーシングに `help` / `h` を有効アクションとして追加
- ヘルプ表示の実装（アクション一覧、短縮形、簡潔な説明）

## 境界
- ヘルプ内容は静的テキスト（動的な状態表示は含まない）
- 各アクションの詳細説明は含まない（一覧のみ）
