## プロンプトファイルの構造的問題

### ステップ番号の不整合

| ファイル | 問題 |
|---------|------|
| inception.md | Part1(1-7)→Part2で番号リセット、5.5/5.6枝番、6-1ハイフン記法混在 |
| construction.md | 3番号体系混在（セットアップ1-5、フローI-VI、完了0-6） |
| operations.md | セットアップ(1-7)/フロー(0-6)二重番号、2.5/2.6/2.7枝番 |
| operations-release.md | 6.4→6.4.2欠番、6.6→6.6.5欠番等 |
| review-flow.md | 5a/5a.1アルファベット記法、深ネスト(6.3.1) |
| commit-flow.md | 2.5/2.6/2.7/3.5枝番 |

### 記法の種類（統一が必要）

- 整数: `1, 2, 3`
- 小数枝番: `5.5, 6.4.2`
- ハイフン: `6-1, 6-2`
- アルファベット: `5a, 5a.1`
