## Epic 2: プロンプト品質改善

### ストーリー 3: ステップファイルの圧縮・分割 (#460)
**優先順位**: Must-have

As a AIエージェント
I want to 全ステップファイルがClaude Codeの Readツール上限以下に収まっている
So that Readツールで一括読み込みでき、初期化オーバーヘッドが削減される

**受け入れ基準**:
- [ ] review-flow.md（現20404トークン）が分割され、各ファイルがReadツールで一括読み込み可能（Claude Codeのトークン上限10000以下）
- [ ] rules.md（現10817トークン）が同上
- [ ] 01-setup.md（現10196トークン）が同上
- [ ] 分割後のファイル間の参照関係が正しく維持されている
- [ ] 分割によって情報の欠落がない

**技術的考慮事項**:
- トークン計測はClaude CodeのReadツールが返すエラーメッセージの値を基準とする
- 冗長な記述の圧縮を優先し、それでも超過する場合に分割
