# Unit: skills/直接参照チェック導入

## 概要
`skills/`以下のファイルでプロジェクトルート相対パス（`skills/aidlc/`）による参照違反をCIで自動検出するスクリプトを作成し、GitHub Actionsに統合する。

## 含まれるユーザーストーリー
- ストーリー 6: skills/直接参照チェック

## 責務
- チェックスクリプトの作成（`bin/check-skill-references.sh`）
- GitHub ActionsのCIジョブへの追加

## 境界
- `.aidlc/rules.md`のファイル参照境界ルール自体は変更しない
- skills/以外のディレクトリは検査対象外

## 技術的考慮事項

- 対象ファイル: `skills/`以下の`.md`, `.sh`, `.toml`ファイル
- コメント内も検出対象
- 終了コード: 違反あり=1、違反なし=0
- 参考実装: `bin/check-bash-substitution.sh`
- CI統合: `.github/workflows/`の既存CIジョブに追加
