## 変換ルール

### 見出し変換

| 変換パターン | 対象 |
|-------------|------|
| H2 → H2（維持） | 不変ルール、制約事項（横断制約）+ 新規グループ見出し |
| H2 → H3（降格） | グループ内に統合されるセクション（13個） |
| H3 → H4 | 親セクション降格で4階層になるもの（質問フロー、情報提示ルール、インタラクション種別等）。H1-H3原則の例外としてグループ化ネスト深化のみH4許容 |

### H3→H4の方針

H2→H3降格により既存H3がH4相当になるケースでは、H4見出しに変換する。H1-H3の3階層原則の例外として、グループ化に伴うネスト深化のみH4を許容する。見出しアンカーとしての参照可能性を維持し、内容・テーブルは一切変更しない:

- `### 質問フロー` → `#### 質問フロー`（内容維持）
- `### 質問不要でも確認が必要な場面` → `#### 確認が必要な場面`（内容維持）
- `### 不明点の記録` → `#### 不明点の記録`（内容維持）
- `### 情報提示ルール` → `#### 情報提示ルール`（内容維持）
- `### インタラクション種別と対応方法` → `#### インタラクション種別と対応方法`（テーブル維持）
- `### セミオートゲート仕様との関係` → `#### セミオートゲート仕様との関係`（内容維持）
