# ドメインモデル: プリフライト圧縮・バージョンチェック外部化

## 概要
preflight.mdのオプションチェック説明をテーブル圧縮し、バージョンチェックロジックをガイドに外部化することでコンテキストサイズを圧縮する。

## エンティティ

### プリフライトチェック手順5（オプションチェック）
- **属性**:
  - ghチェック: ghステータスの判定と表示
  - レビューツール確認: CLI存在確認と表示
  - 設定バリデーション: defaults.toml存在確認
- **振る舞い**: 散文説明をテーブル形式に圧縮

### バージョンチェック（三角モデル）
- **属性**:
  - バージョン情報取得（3点: リモート、スキル、ローカル設定）
  - ComparisonMode決定（5モード）
  - 比較実行（各モードのパターンテーブル）
- **振る舞い**: 01-setup.mdからguides/version-check.mdに移動

## ユビキタス言語
- **オプションチェック**: blocker以外の環境チェック項目（gh, レビューツール, defaults.toml）
- **三角モデル**: リモート・スキル・ローカル設定の3点バージョン比較方式
- **ComparisonMode**: 利用可能なバージョン情報の組み合わせに基づく比較モード
