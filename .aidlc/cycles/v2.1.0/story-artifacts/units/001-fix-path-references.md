# Unit: パス参照問題の調査・修正

## 概要
プラグイン環境で破綻する `../../` 等の相対パス参照を修正し、スキル間の内部依存を除去する。

## 含まれるユーザーストーリー
- ストーリー 1: パス参照問題の解消

## 責務
- `skills/` 配下のスクリプト・Markdownファイルの `../../` 参照を調査・修正（`scripts/tests/` 配下はメタ開発固有の例外として除外）
- aidlc-migrate の `migrate-detect.sh` のプラグイン環境対応
- aidlc-setup のスクリプトのパス参照修正
- aidlc guides 内のMarkdownリンク修正
- スキル間の内部ファイル直接参照の除去

## 境界
- `scripts/tests/` 配下のテストスクリプトはメタ開発固有の例外として修正対象外
- `review-flow.md`, `rules.md` のパス参照・スキル名変更は Unit 002 の責務（本Unitでは対象外）
- レビュースキルの再構成は Unit 002 の責務

## 技術的考慮事項
- `SCRIPT_DIR` ベースの相対パスをスキルベースディレクトリ相対に変更
- `AIDLC_PLUGIN_ROOT` 環境変数の活用を検討
- Markdownリンクはスキルベース相対パスに統一
